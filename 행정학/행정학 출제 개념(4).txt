04. 인사 행정론
1> 인사행정 기초이론
 * 전략적 인적자원관리

 * 엽관주의
  - 임용 기준: 정치적 충성심
  - 주요 내용
   = 대통령 선거
   = 정권 교체
   = 공직경질제
  - 주요 장점
   = 공직의 특권화 배제
   = 행정의 민주성, 대응성
  - 주요 단점
   = 능률성 저하
   = 부패, 부질서 만연
  - 추구하는 가치
   = 기본적 가치: 민주성, 형평성
   = 수단적 가치: 정치적, 정당적 대응성
 * 실적주의
  - 임용 기준: 개개인의 능력, 자격, 적성, 시험
  - 주요 내용
   = 시험
   = 정치적 중립성
   = 신분보장
  - 주요 장점
   = 공직임용의 기회균등
   = 행정의 능률성
  - 주요 단점
   = 관료주의, 관료 특권화, 보수화
   = 정당이념 및 정책의 실현, 추진 곤란
  - 추구하는 가치
   = 기본적 가치
   = 수단적 가치

 * 직업공무원제
 - 개념
  = 직업공무원제란 공직이 유능하고 젊은 인재에게 개방되어 있고 명예로운 높은 지위로 승진하는 기회가 보장되어 공직근무를 보람있는 생애로 생각하고 평생을 공직에 바치도록 조직, 운영되는 공무원제도 말함
 - 특징
  = 행정의 독자성과 안정성, 계속성, 능률성 확보
  = 정치적 변동 시에도 공무원이 중립적 권력으로서 국가 전체의 통일성, 항구성을 유지 확보하게 해줌
  = 전문화 경향에 대응할 수 있음
 - 장점
  = 행정의 안정성, 계속성 유지
  = 행정의 중립성 확보
  = 유능한 인재 유치
  = 동기 부여
  = 강한 전문 직업의식, 공복의식
 - 단점
  = 행정의 전문화 저해
  = 관직의 폐쇄화, 전문가의 임용 곤란
  = 민주통제의 곤란
  = 환경에 대한 적응력 부족
  = 공직임용 기회의 불균등 초래
  = 공직 이외의 직업 전환 곤란

 * 대표관료제
  - 개념
   = 대표관료제는 킹슬리가 처음 사용한 용어로, 실적주의 인사제도의 폐단을 극복하기 위해 등장
   = 인종, 종교, 성별, 신분, 계층, 지역 등의 여러 기준에 의하여 분류되는 모든 사회집단들이 인구 전체 안에서 차지하는 비율에 맞게 관료조직의 직위들을 차지해야 한다는 원리가 적용되는 관료제
   = 정부관료제가 그 사회의 인적 구성을 반영하도록 구성함으로써 관료제 내의 민주적 가치를 주입하려는 의도에서 발달된 개념이며 임명직 관료집단이 민주적 방법으로 행동하도록 하기 위한 방안으로 도입
  - 전제
   = 출신성분과 인간의 행동 간에는 밀접한 관련성이 있음을 전제. 자기 출신집단의 가치와 이익을 정책결정에 반영히키는 데 노력하고, 자기 출신집단의 가치와 이익을 정책결정에 반영시키는 데 노력하고, 자신이 선호하는 개인적 정책가치의 극대화를 추구하며, 정책관료의 가치관, 태도가 출신집단 똔느 일반 국민의 가치괸, 태도에 유사하면 정책의 대응성이 보다 더 제고된다는 것을 전제함
   = 배경적 대표성이 태도적 대표성으로 이어짐. 실질적 대표성을 낳는다는 논리에 기초함
   = 임용할당제는 관료제의 비례대표성을 확보하기 위한 방안의 하나
  - 필요성
   = 정부관료제의 대표성 강화와 대응성, 책임성 제고
   = 내부통제의 강화와 책임성 제고
   = 기회 균등의 적극적, 실질적 보장과 수직적 형평성 제고
   = 실적주의적 역기능 시정
  - 문제점
   = 실적제에 대한 갈등
   = 행정의 전문성과 능률성, 생산성 저하
   = 재사회화의 불고려
   = 역차별의 우려
   = 사회적 형평성 저해

 * 성과주의 인적자원관리
 
 
 * 중앙인사기관
  - 개념
   = 정부의 인사행정을 전문적으로 연구하고 인사정책을 수립하며, 그 집행을 총괄하는 기관
  - 목적
   = 공무원의 수적 증가로 합리적 인사관리를 위한 기관 설치 필요
   = 인사행정의 공정성, 중립성 확보 필요
   = 전문화, 눙률화를 추진하는 데는 집권적인 인사기관이 요구
   = 지나친 분산, 할거성 규제하고 인사행정의 효율적인 조정, 통제와 인사행정의 통일성 가하는 기관 필요
  - 기능
   = 준입법적 기능
   = 준사법적 기능
   = 집행기능
   = 감사 및 자문기능
   = 기타

 * 소청심사위원회
 - 인사혁신처에 부속된 합의제기관으로 직무상의 독립성 보장
 - 징계처분이나 강임, 휴직, 직위해제, 면직처분과 같이 그의 의사에 반하는 불이익 처분에 이의를 제기하는 경우, 이를 심사해 구제하는 걸차
 - 위원장 1명을 포함한 5명 이상 7명 이하의 상임위원과 상임위원 수의 1/2 이상이 비상임위원으로 구성되며 그 결정에 법적 구속력이 수반됨. 소청심사위원회의 결정은 처분행정청을 기속함
 
2> 공직 분류
 * 개방형 직위
  - 공직의 모든 계급이나 직위를 불문하고 공직 내외의 모두로부터 신규채용이 허용되는 인사체계
  - 장점
   = 인사행정의 적극화에 의한 우수한 인재 등용 유리
   = 공무원 및 행정의 질적 향상 기여
   = 조직의 신진대사 촉진하고 공직의 침체 방직
   = 무사안일주의, 관료주의 방지
   = 관료에 대한 민주통제가 확보됨
   = 충원, 임용의 융통성이 증대됨
   = 기관장의 적절한 영향력과 리더십이 발휘될 수 있음, 고위층의 조직장악력을 강화
   = 전문성이 요구되는 경우 일정한 직무수행요건을 갖춘 자를 공직 내외부에서 임용하여 공직 전문성 제고 가능

 * 직위분류제
  - 개념
   = 직무의 종류와 곤란성 및 책임성의 기준으로 공직을 분류하는 제도 말함
   = 미국, 호주, 캐나다, 필리핀 등에서 채택함
   = 모든 대상 직위를 직무의 종류와 곤란성 및 책임도에 따라 직군, 직렬, 직급 또는 직무등급별로 분류하되, 같은 직급이나 같은 직무등급에 속하는 지위에 대하여는 동일하거나 유사한 보수가 지급되도록 분류하여야 함.
  - 특징
   = 특정 직무의 수행능력 중요시
   = 자유로운 충원이 이루어지는 개방형 채택
   = 직위분류제긔 기본구조인 등급은 제도상으로는 사회적 신분, 지위를 나타내는 것이 아니므로, 상위직-하위직 간의 대립의식, 위화감이 크지 않음
   = 능률주의적, 과학적 인사행정의 기반을 제공함
  - 수립 절차
   = 계획 수립
   = 분류담당자 선정, 분류대상 직위의 설정
   = 직무조사-직무기술서의 작성
   = 직무분석
   = 직무평가
   = 직무명세서 작성
   = 정급
   = 사후 검토
  - 장점
   = 보수체계의 합리화
   = 적재적소의 인사배치
   = 행정의 전문화, 분업화에 촉진
   = 시험의 합리화
   = 훈련수요의 명확화
   = 근무성적평정의 객관적 기준 제시
   = 권한, 책임한계의 명확화
   = 예산행정의 능률화와 행정의 민주적 통제
   = 효율적인 정원관리
   = 사무관리의 개선
   = 정부의 홍보에 도움
  - 단점
   = 인사배치의 신축성, 융통성의 결여
   = 유능한 일반행정가 확보 곤란
   = 장기적, 종합적 능력 발전과 직업공무원제의 확립 곤란
   = 단기적 효용
   = 커뮤니케이션, 협조, 조정의 곤란
   = 신분 불안
   = 사무적 인간관계

 * 계급제
  - 개념
   = 사람을 중심으로 개개인의 일반적인 능력과 자격을 기준으로 공무원을 계급으로 분류하는 제도
   = 계급제하에는 인적자원 활용의 수평적 융통성은 높으나 수직적 융통성은 낮음
   = 농업사회의 전통을 지닌 영국, 프랑스, 독일, 일본 등에서 채택함
   = 계급제 중심의 공직분류형에 직위분류제 요소를 가미하고 있음
  - 특징
   = 교육제도상의 계층과 밀접한 관계가 있는 4대 계급제가 확립
   = 계급 간의 차이가 심하며 승진 어려움
   = 고급공무원의 수가 적고 높은 학력이 요구되고 있으므로, 고급공무원의 엘리트화 현상이 현저하게 나타남
   = 폐쇄형의 인사관리방식이 채택됨
  - 장점
   = 일반적 교양과 능력 있는 자의 채용
   = 인사배치의 신축성, 융통성
   = 유능한 일반행정가의 양성
   = 장기적, 종합적 능력 발전 및 직업공무원제의 발전 촉진
   = 장기적 효용성
   = 커뮤니케이션, 협조, 조정의 원활
   = 신분보장의 강화
  - 단점
   = 행정의 전문화 및 전문행정가의 양성 어려워짐
   = 권한, 책임 불분명
   = 인사관리의 객관적 합리화 및 객관적 기준 설정 곤란
   = 인력수급계획의 수립 곤란
   = 강한 서열의식이 지배되어 상-하간 의사소통 장애 초래
   = 신분조상과 폐쇄형 임용체제로 인해 무사안일 부추기고 공무원의 특권화 유발

 * 경력직 공무원
  - 일반직
   = 기술, 연구, 행정일반에 대한 업무 담당하는 공무원
   = 1급부터 9급까지의 계급으로 구분, 직업공무원의 주류
   = 특수업무, 연구, 지도, 특수기술 직렬, 그리고 특별한 경우에 한해 계급 구분이나 직렬 구분이 없음
  - 특정직 공무원
   = 별도의 법률에 의해 별도의 직급체계가 있음
   = 국가직: 법관, 검사, 외무, 경찰, 소방, 교유, 군인 공무원 등
   = 지방직: 공립대, 전문대의 교육공무원, 
 * 특수경력직 공무원
  - 국가공무원법이나 실적주의, 직원공무원제의 획일적 적용을 받지 않으면서, 정치적 임용이 필요하거나 특수한 직무를 담당하는 자. 계급 구분이 없음.
  - 정무직 공무원
   = 취임하거나 임명하는데 국회 동의 필요
   = 고도의 정책결정업무를 담당하거나, 업무를 보조하는 공무원으로서 법률 또는 대통령령으로 지정함.
   = 장-차관, 감사원장과 사무총장, 국가정보원 원장과 차장, 국회사무총장, 지방의회 의원
  - 별정직 공무원
   = 별도의 자격기준에 의하여 임용되는 공무원으로서, 법령에서 별정직으로 지정하는 공무원
   = 국가정보원 기획조정실장, 국회수석전문위원 등이 이에 해당함  
 
 * 정무직
  - 선거에 의하여 취임하거나 임명하는 데 국회의 동의를 요하는 공무원
  - 고도의 정척결정업무를 담당하거나 업무를 보조하는 공무원으로서 법률 또는 대통령령에서 정무직으로 지정하는 공무원임
  - 장,차관, 감사위원장과 사무총장, 국가정보원 원장과 차장, 국회사무총장, 지방의회 의원 등이 해당

 * 직무평가방법
  - 서열법
   = 사용빈도: 가장 적음
   = 비교방법: 직무와 직무
   = 요소의 수: 없음
   = 척도의 형태: 서열
   = 다른 방법과의 관계: 요소비교법의 조잡한 형태
   = 평가방법: 비계량적 방법
   = 평가대상: 직무 전체
  - 분류법
   = 사용빈도: 둘째나 셋째
   = 비교방법: 직무와 기준표
   = 요소의 수: 없음
   = 척도의 형태: 등급
   = 다른 방법과의 관계: 점수법의 조잡한 형태 
   = 평가방법: 비계량적 방법
   = 평가대상: 직무 전체
  - 점수법
   = 사용빈도: 가장 많음
   = 비교방법: 직무와 기준표
   = 요소의 수: 평균 11개
