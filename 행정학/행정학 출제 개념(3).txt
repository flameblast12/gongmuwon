03. 조직 이론
1> 조직 기초이론
 * 조직이론과 유형
  - 고전적 조직이론
   = 기초이론: 과학적 관리론
   = 인간관: 합리적 경제인관
   = 추구하는 가치: 기계적 능률
   = 주 연구대상: 공식적 구조
   = 환경: 폐쇄형
   = 연구방법: 원리 접근법
  - 신고전적 조직이론
   = 기초이론: 인간관계론
   = 인간관: 사회인관
   = 추구하는 가치: 사회적 능률
   = 주 연구대상: 비공식적 구조
   = 환경: 폐쇄형
   = 연구방법: 경험적 접근법
  - 현대적 조직이론
   = 기초이론: 체제이론
   = 인간관: 복잡인관
   = 추구하는 가치: 다원적 가치
   = 주 연구대상: 동태적, 유기적 구조
   = 환경: 개방형
   = 연구방법: 복합적 접근법

 * 신고전적 조직이론
 - 1930년대, 인간중심
  - 호손 실험과 인간관계론에 이론적 근거를 두고 성립, 발전됨
  - 고전적 조직이론과는 달리 행정조직 내의 비공식적, 비합리적, 정서적, 사회적, 심리적 측면의 중요성 강조
  - 특징
   = 사회적 능률성 추구
   = 사회적모형으로서의 조직관
   = 인간관계를 중시
   = 사회적 인간 모형
   = 실증주의, 인간주의

 * 고전적 원리
  - 과학적 관리법의 영향하에 공식적, 합리적 조직에 중점을 둠
  - 환경과의 관계를 고려하지 않으며, 조직의 원리, 부처의 편성 기준, 관리자의 가능 등을 중시함
  - 특징
   = 능률지상주의
   = 기계적 조직관
   = 원리주의적 접근법 중시
   = 합리적, 경제적 인간모형
   = 구조, 기술 중심의 행정개혁
   = 공-사행정 일원론
   = 수단 중시
   = 과학적 관리론

 * 거시조직이론
  - 결정론: 개별 조직 하나하나를 미시적 수준으로 보는 것
   = 개별조직: 체제구조적 관점, 구조적 상황이론
   = 조직군: 자연적 선택 관점, 조직군 생태학 이론, 조직경제학.
  - 임의론: 조직 전체를 거쳐 거시적 수준으로 보는 것
   = 개별조직: 전략적 선택 관점, 전략적 선택이론, 자원의존이론
   = 조직군: 집단적 행동 관점, 공동체 생태학 이론
  
 * 상황론적 조직이론
  - 상황적응이론
   = 조직의 환경적 요인을 강조하면서 고도의 불확실한 상황에서 최선의 관리방법은 없으며, 다만 효과적인 방법이 있다는 말
  - 특징
   = 최선의 방법 부인
   = 효과적 방법 인정
   = 조직과 환경의 관계 중심
   = 객관적 결과 중시
   = 분석단위
   = 중범위이론 지향
  - 결론
   = 관료제와 애드호크라시를 서로 보완적인 관계로 보고, 상호유기적인 조화를 이루고자 함.
   = 동질적이고 안정된 상황: 정형화되고 계층적인 조직
   = 다양하고 변동하는 상황, 환경: 형식화, 정형화 정도가 덜하고 유기적인 조직

 * 조직군생태 이론
  - 환경에 따른 관점: 통제 불가능하며 단순히 주어짐
  - 적응 방식: 환경에 의한 선택
  - 분석 수준: 개체군(비교적 동질적인 조직들의 경제적 집합체)
  - 조직 간 관계: 경쟁적
  - 관리자 역할: 무기력, 상징적
  - 학문적 바탕: 개체군 생태학

 * 거래비용이론
 - 거래비용은 시장과 기업 가운데 어떠한 방식이 어떠한 경우에 좀 더 효율적일 수 있는가를 분석하는 데 연구의 초점이 있음.
   = 불확실성이 대단히 높은 상황에서 내부조직은 의사결정자의 제한된 합리성을 경제화하여 주는 존재라는 것
  - 거래비용이론은 대리인이론을 조직이론에 적용한 것으로, 의사결정에 따른 거래비용이론을 최소화하기 위한 조직화의 원리를 추구함. 조직은 경제활동에서 재화나 용역의 거래비용을 최소화하기 위한 조직화의 원리 추구. 조직은 경제활동에서 재화나 용역의 거래비용을 줄이기 위해 만들어지는 장치
  - 거래비용이론은 시장에서 이뤄지는 개인 및 조직 간 거래를 미시적으로 분석하며, 거래비용은 통제비용, 거래관례의 유지비용, 정보비용 등 경제적 교환과 관련된 모든 비용을 의미. 거래비용이론에서는 자산의 특정성과 정보의 편재성, 거래빈도 등이 거래비용을 증가시키는 요인
  - 전속적 투자가 높을 경우 시장거래보다 기업 내부조직 거래를 관계적인 계약으로 선호하게 됨. 시장의 자발적인 교환행위에서 발생하는 거래비용이 관료제적 조정비용보다 더 크면 최소화를 위하여 거래의 내부화, 즉 조직의 통합이 이루어진다는 이론
  - 거래비용을 극소화하기 위해 종전의 U형(단순형)에서 M형(다차원적) 관리로 전환할 것을 주장함. 사업부제 조직모형인 M형 구조가 U형 구조보다 효율적이라고 주장
  - 동일한 업무를 기업 안 또는 바깥에서 처리할 때의 조직관리비용/거래비용을 비교하여 해당 업무를 어디서 수행할지를 결정하는 이론임
  
 * 자원의존이론
  - 주요 내용
   = 페퍼와 살랜시크에 의하여 조직의 유효성 증대라는 최종 목표를 위해서는 조직의 환경에 대한 자원 의존도와 불확실성을 감소시키는 것이 필요하다는 입장
   = 자원의존이론은 어떠한 조직도 외부환경으로부터 필요로 하는 다양한 모든 자원을 획득할 수 없다는 것을 전제, 조직이 환경적 요인을 피동적으로 받아들이지 않고 스스로의 이익을 위하여 적극적으로 환경에 대처하며 조직 내의 대내적, 정치적 맥락에서 조직의 환경적응을 위한 전략적 결정을 내린다고 봄. 즉 조직은 환경에 대한 의존 정도의 변화를 통하여 영향을 미칠 수 있다고 보는 것
  - 상황이론과의 비교: 조직이 환경에 의하여 영향을 받는다는 점에서 상황이론과 동일하지만, 환경을 변화시킬 수 있다고 보는 점에서 상황이론과 다름
  - 전략적 선택이론과의 비교: 인지에 의하여 달라질 수 있다는 전략적 선택이론과는 달리, 자원의존이론은 인지에 의하여 달라질 수 있다는 전략적 선택이론과는 달리, 자원이론에서는 환경은 여전히 조직의 활동에 영향을 미치고 있는 제약이 됨. 가급적 환경에 대한 의존 정도를 변화시키고자 노력하는 점에서 차이
 
 * 조정방법
  - 조직의 제 원리에 의한 방법
  - 권한 및 책임 한계의 명확화
  - 목표에 의한 조정
  - 이념에 의한 조정
  - 조정기구의 설치
  - 회의, 위원회에 의한 방법
  - 사전 계획 및 사후 환류에 의한 조정
  - 절차의 정형화, 규칙
  - 자율적인 조정
  - 인사에 대한 배려
 * 조정기제
  - 수직연결: 계측제, 규칙과 계획, 계층 직위의 추가, 수직정보시스템 등
  - 수평연결: 정보시스템, 직접접촉, 임시작업단, 프로젝트매니저, 프로젝트팀, 연락담당자 등
 
 * 계층제
  - 개요
   = 직무를 권한과 책임의 정도에 따라 등급화, 계층화하고 상하계층 간에 지휘, 명령복종 관계를 확립하는 것을 말함
   = 오늘날 관료제 조직은 피라미드형 수직적인 계층제 확립
   = 계선조직 중심으로 형성
  - 특징
   = 조직의 규모와의 관계
   = 조직의 전문화의 관계
   = 통솔범위의 관계
   = 업무의 성격
   = 관료제 병리와의 관계
  - 순기능
   = 지휘, 명령의 통로, 상하간 커뮤니케이션 통로
   = 행정목표 설정 통로, 업무배분 통로
   = 권한의 체계적 위임
   = 내부통제와 조정의 효율적 수단
   = 행정 능률성과 책임의 명확성 보장 수단
   = 승진 경로, 동기부여, 사기앙양
  - 역기능
   = 계층이 심화될수록 커뮤니케이션 장애와 왜곡이 초래되기 쉬움
   = 조직의 경직성을 초래하여 환경변동에 신축성 있게 적응하기 어려움
   = 인간을 지배하는 신분상의 상하관계로 오인되기 쉬워 동태적이고 융통성 있는 인간관계의 형성을 저해
   = 집단기능보다 개인에게 의존하게 되어 기관장의 독단화가 우려되며 조직구성원의 쇄신적, 창조적 활동 어려움
   = 자율성이 강한 전문가는 계층제의 권위와 대립, 갈등을 빚음
  = 인간의 자아실현 욕구와 잘 조화되지 않음
  = 할거주의를 초래하기 쉬움 


2> 조직구조론
 * 기본변수, 상황변수
  - 기본변수
   = 복잡성
   = 공식화
   = 집권화
  - 상황변수
   = 조직규모
   = 조직기술
   = 환경

 * 기계적 구조 - 예측가능성
  - 조직 특성
   = 좁은 직무 범위
   = 표준운영절차
   = 분명한 책임관계
   = 계층제
   = 공식적, 몰인간적 대면관계
  - 상황 조건
   = 명확한 조직목표와 과제
   = 분업적 과제
   = 단순한 과제
   = 성과 측정이 가능
   = 금전적 동기부여
   = 권위의 정당성 확보
 * 유기적 구조 - 적응성
  - 조직 특성
   = 넓은 직무 범위
   = 적은 규칙과 절차
   = 모호한 책임관계
   = 분화된 채널
   = 비공식적, 인간적 대면관계
  - 상황 조건
   = 모호한 조직목표와 과제
   = 분업이 어려운 과제
   = 복합적 과제
   = 성과 측정이 어려움
   = 복합적 동기부여
   = 도전받는 권위
 
 * 기능구조 
  - 장  = 각 부서  = 부서의 하위 부서
(1) 개요
  - 기능구조는 각 기능부서화 방식에 기초한 조직구조 유형으로, 조직의 전체 업무를 공동기능별로 부서화
  - 기본적으로 수평적 조정의 필요가 낮을 때 효과적인 조직구조, 특정 기능에 관련된 조직구성원들의 지식과 기술이 통합적으로 활용되므로 그 전문지식과 기술의 깊이를 제고할 수 있는 조직구조
  - 기능구조는 조직목표달성에 깊은 전문지식이 필수적인 경우, 안정적 조직환경과 일상적 조직기술, 그리고 조직이 수직적인 계층제로 통제, 조정될 필요가 있는 경우, 수평적인 조정의 필요가 적은 경우, 내정 능률성이 중요한 경우에 효과적인 조직구조.
   = 집권적 권한, 높고 전문화된 직무, 부서 간 팀워크가 적게 요구된다는 특면으로 기계적 구조에 속함
(2) 장, 단점
  - 장점
   = 규모의 경제 제고, 같은 기능적 업무를 묶어 시설과 자원을 공유함으로써 중복과 낭비를 막아 효율성 높일 수 있는 것
   = 비슷한 기능을 수행하는 조직구성원들 간의 분업을 통해 전문기술을 더욱 발달시킬 수 있음
   = 비슷한 기술과 경력을 가진 조직구성원들 간에 응집력이 강화되어 부서 내 의사소통과 조정이 유리해짐
  - 단점
   = 조정과 협력이 요구되는 환경 변화에 둔감. 부서별로 상이한 기능을 수행하며 각각 독특한 사관, 목표관을 갖게 되어 부서들 간의 조정 어려움. 할거주의 유발 우려
   = 의사결정 권한이 고위관리자에 집중되어 업무의 과부하에 걸려 빠르게 대처하지 못함
   = 고도의 전문화에 따라 일상적이고 동기부여에 분리

 * 사업구조
  - 장  = 생산물 1, 2  = 각 생산물의 하위 부서들
(1) 개요
  - 사업구조는 산출물에 기반한 사업부서화 방식의 조직구조 유형으로 산출물구조, 사업구조, 전략사업 단위라고도 함
  - 각 부서는 한 제품을 생산하거나, 한 지역에 봉사하거나, 또는 특정적 단위이므로, 기능 간 조정이 극대화될 수 있는 조직구조
  - 다른 시장에 대한 상품을 생산하는 대규모 기업의 경우 사업부서들은 자율적으로 운영, 기능의 조정이 부서 내에서 이루어지므로 기능구조보다 분권적인 조직구조를 가짐
  - 사업구조는 기능 간 조정이 우수하므로 불확한 조직환경, 비정규적인 조직기술, 높은 부서 간 상호의존성, 외부지향적 조직목표인 상황에 적합한 조직구조임
(2) 장, 단점
  - 장점
   = 사업구조의 각 부서는 자기완결적 기능단위로 기능 간 조정이 용이하므로 환경 변화에 좀 더 신축적이고 대응력 높일 수 있음
   = 특정 산출물별로 운영되기 때문에 고객만족도를 제고할 수 있음
   = 성과에 대한 책임성의 소재가 더욱 분명해져 성과관리 유리
   = 조직구성원들에게 기능구조보다 포괄적인 목표관 갖게 해줌
  - 단점
   = 산출물별 생산라인의 중복에 따른 규모경제와 효율성 손실
   = 기능직위가 부서별로 분산되므로 기술적 전문지식과 기술발전에 유리
   = 사업구조의 부서 내 조정은 증진되지만, 자율적으로 운영되는 부서 간의 조정으 더욱 어려워짐
   = 사업무서 간의 경쟁이 지나치면 조직 전반에 부정적 결과 초래
  
 * 매트릭스구조
  - 장 아래에 각 부처와 각 생산물이 표처럼 됨
(1) 개요
  - 매트릭스구조란 기능구조와 사업구조의 화학적 결합을 시도하는 조직구조
  - 조직환경이 복잡해지면서 기능부서의 기술적 전문성이 요구되는 동시에 사업부서의 신속한 대응성의 필요가 증대되면서 등장한 조직 형태
   = 기능구조는 전문가의 집합으로 전문성을 살릴 수 있으나 조정이 어렵고, 사업구조는 전문가의 조정은 용이하나 비용이 중복된다는 문제가 있어 양자의 장점을 채택한 조직구조
(2) 특성
  - 이원적 권한체계
  - 기능부서 통제 권한의 계층은 수직적으로 흐르고, 사업부서 간 조정 권한의 계층은 수평적으로 흐름. 이중구조에서 조직구성원은 동시에 두 명의 상관에 보고하는 체계를 가짐. 명령통일의 원리에 위배되며 기능적, 사업적 권한 체제의 균형을 찾는 것이 중요한 문제가 됨
  - 두 상관의 갈등적인 요구를 해결해야 하는 매트릭스조직의 구성원은 탁월한 인간관계기술이 필요. 매트릭스구조의 상관은 부하에 대해 완전한 통제력을 갖지 못하며, 상관들 간의 대면, 협력, 갈등을 조정할 수 있는 관리능력 요구
  - 다음과 같은 조직에 알맞은 구조
   = 생산라인 간에 부족한 자원을 공유해야 할 압력 존재
   = 두 개 이상의 핵심적 산출물에 대해 품질성과 수시적 제품 개발의 압력이 있을 경우
   = 조직의 환경영역이 복잡하고 불확실한 경우
(3) 장, 단점
  - 장점
   = 신축성과 적응성이 요구되는 불안정하고 급변하는 조직환경에 효과적인 구조
   = 잦은 대면과 회의를 통해 예상치 못한 문제 파악. 새로운 해결책 찾는데 기여할 수 있음
   = 구성원들을 부서 간에 공유함으로써 조직을 자원의 호율성을 제고함
   = 개인들은 다양한 경험을 통해 전문기술 개발과 함께 더 넓은 시야와 목표관을 갖게 할 수 있는 기회 가짐. 조직구성원 직무 동기부여에 기능적으로 작용
  - 단점
   = 이중권한 체제가 개인에 미치는 혼란, 갈등, 좌절의 가능성 높음
   = 기능부서와 사업부서 간의 갈등의 가능성 생김
   = 갈등해결에 요구되는 시간과 노력 낭비 불가피
   = 기능부서와 사업부서 간의 적절한 권력균형 찾는 것이 어려움

 * 수평구조(팀구조)
(1) 개요
  - 개념
   = 새로 등장한 유기적 구조모형의 하나는 수평구조 또는 팀구조이며, 조직구성원을 핵심업무과정 중심으로 조직하는 방식
   = 특정한 업무과정에서 일하는 개인을 팀으로 모아 의사소통과 조정을 쉽게 하고, 고객에게 직접적으로 가치를 제공하도록 함
  - 등장 배경: 조직구조 접근법은 최근의 새로운 사업환경 변화에 대한 대응으로 등장하게 됨. 기술의 진보로 정보통신기술에 기초한 조정과 통합이 가능해졌으며, 고객은 더욱 빠르고 좋은 서비스 기대. 새로운 기술을 학습하고 좀 더 큰 책임감 감수
(2) 특징
  - 부서 간 조정을 위한 교차기능팀, 임시과제 수행을 위한 임시작업단(태스크포스)등 수평적 기제 실험. 1990년대 이후에는 리엔지니어링의 결과 상설 팀제를 조직 전반에 도입하는 사례 적지 않게 확인. 수직계층과 부서 간 경계를 실질적으로 제거하는 대단히 유기적인 조직구조인데, 특성은 다음과 같음
   = 과업, 기능, 지리에 기반하지 않고 핵심과정에 기초
   = 수평구조의 기본 구성단위는 자율팀으로 과업수행에 필요한 자원에 접근할 수 있고, 의사결정 권한 가짐
   = 핵심과정에 대한 전체적인 책임은 각 과정조정자가 짐
   = 팀원은 여러 직무를 수행할 수 있게 훈련 받음
   = 조직효과성은 핵심과정별 최종성과지표, 고객만족도, 종업원만족도, 재정기여도 등에 의해 평가
(3) 장점
  - 고객수요 변화에 신속히 대응할 수 있는 조직의 신축성을 크게 제고할 수 있음. 구조 자체가 모든 구성원들의 관심을 고객만족도의 증진에 두게 됨
  - 부서 간 경계가 없어 개인들은 조직 전체의 관점에서 업무를 이해, 팀워크와 조정에 유리한 조직
  - 자율관리, 의사결정, 권한과 책임을 위임함. 사기와 직무동기부여에 기여
(4) 방향
  - 수평구조는 먼저 조직목표달성에 핵심적인 업무과정을 분석해야 하는 준비
  - 수평구조로 변화하는 데에는 직무설계, 관리철학, 정보 및 보상체계 등에 대한 대대적인 혁신을 요구하는 장기적이고도 어려운 준비과정 필요
  - 관리자는 권력을 행사하는 감독자에서 팀을 후원하는 코치로 역할 변화를 인정하고, 부하들은 자율적이고 신축적인 업무환경에서 일할 준비가 되어야 함.
 
 * 네트워크구조
  - 장 아래에 여러 생산물
(1) 개요
  - 조직의 자체 기능은 핵심역량 위주로 합리화, 여타 기능은 외부기관들과 계약관계를 통해 수행하는 조직구조 방식
  - 유기적 조직유형의 하나로서, 정보통신기술의 확산으로 채택된 새로운 조직구조 접근법
  - 한 조직 내에서 모든 기능을 수행하는 방식 탈피, 회계, 제조, 포장, 유통 등을 외부기관에 아웃소싱하고 전략수립과 판매는 직접 담당하는 등의 예시. 환경 변화에 따라 신속하게 외부계약 관계를 재조정, 정보통신기술의 속도와 편리성이 필수적인 기반시설
(2) 장, 단점
  - 장점
   = 전지구적으로 최고 품질과 최저비용의 자원들을 활용할 수 있으면서도, 대단히 간소화된 조직구조를 갖는데 있음. 정보통신망에 의해 조정되므로 직접 감독에 필요한 많은 지원, 관리인력 불필요
   = 환경 변화에 신축적이고 신속한 대응 가능. 경쟁력 있는 기술을 가진 활동 집중. 나머지 활동은 계약 관계로 수행하여 환경 변화에 따른 거대한 초기 투자 없이도 새 제품 신속히 출신 가능
   = 내 개인들은 도전적인 과업을 수행하면서 직무의 확장과 확충에 따라 직무동기가 유발
   = 구성단위의 자율성이 높으며 관계가 수평적. 수평적, 공개적 의사전달 강조
   = 조직망 속의 중심점들 간의 지속적인 교환관계에서 정보의 새로운 종합과 지식의 산출 증진시킴으로 학습 촉진
  - 단점
   = 외부기관을 직접 통제하기 어렵다는 점. 대리인 문제가 발생하기 쉬워, 조정과 감시비용 증가
   = 안정적 공급과 품질관리의 어려움
   = 모호한 조직경계에 따라 조직의 정체성이 약해 응집성 있는 조직문화를 가지기 어렵고, 충성을 기대하기 쉽지 않음. 조직구성원들이 자주 이직하는 경향

 * 애드호크라시
  - 개념
   = 행정조직의 동태화란 환경 변화에 신축성 있게 적응하고 끊임없이 제기되는 행정수요를 충족시킬 수 있도록, 경직화된 수직적 구조의 행정조직을 변동 대응능력을 가진 쇄신적 조직으로 전환시켜 문제해결 중심의 협동체제를 구성해 나가는 화정 의미함
   = 관료제의 결함을 보완하고자 등장한 임시적, 동태적 유기적 조직을 총징함.
  - 구조적 특징
   = 높은 수준의 수평적 분화, 직무 전문화. 낮은 수준의 수직적 분화, 직무 전문화
   = 낮은 수준의 집권화, 분권적, 전문적 의사결정
   = 높은 수준의 전문화로 낮은 수준의 공식화, 표준화 강조
   = 고도의 유기적 구조
  - 장점
   = 적응성, 창조성 제고
   = 다양한 전문가 간의 유기적인 협조가 확보될 수 있음
   = 비정형적 업무, 고도의 전문적 업무, 상호의존도가 높은 업우에 좀 더 유용함
  - 단점
   = 명확한 서열 구분이 없어 구성원 간에 갈등과 비협조 불가피
   = 책임, 과업의 불명확성으로 책임 의식 약화
   = 임시성에 따른 심리적 불안감 야기
   = 비효율적인 구조이며 기계적 편의성, 정밀성 결여되어 있음
   = 창의적인 업무수행과정에서 직원들이 심적 스트레스를 많이 받음
  - 주요 형태
   = 프로젝트팀: 특정 사업, 과제 수행을 위해 조직 내의 관련 전문가 중심으로 결합되는 수평적, 임시적, 동태적 조직
   = 태스크포스: 특정 목적, 임무를 서행하기 위해 여러 조직의 전문가를 결합하는 입체적, 임시적, 동태적 조직
   = 매트릭스조직: 전통적인 기능구조 프로젝트팀을 혼합함으로써 수직적 구조와 수평적 구조가 혼합 편성된 임시적, 동태적 조직.

 * 갈등
  - 개념
   = 사이몬은 갈등이란 의사결정과정에서 대안의 선택기준이 명확하지 못하여 개인이나 집단, 조직이 대안의 선택에 심리적 곤란을 겪는 상황을 의미하는 것으로 봄
   = 갈등이란 일반적으로 인간의 심리적 대립이나 대립적 상호작용, 행동을 가리키는 매우 다양성을 띈 개념
  - 기능의 변천
   = 전통적 견해(1940년대 중반 이전): 갈등을 부정적으로만 보고 조직관리자들은 그러한 갈등의 찾아내서 그것을 제거하는 것이 조직과 집단의 성과를 개선하는 길이라고 봄
   = 행태주의적 견해(1940년대 후반~1970년대 중반): 갈등이란 조직 내에서 필연적으로 발생할 수 밖에 없는 현당으로, 이를 완전히 제거한다는 것은 불가능하고 때로는 갈등이 집단의 성과를 향상시킨다고 봄
   = 상호작용주의적 견해(1970년대 중반 이후): 갈등이 오히려 조직 내에서 추진력으로 작용할 수도 있다는 것을 제시, 갈등의 형태에 따라 바람직하지 못하는 것과 바람직한 것을 구분하여 조직의 목표 달성에 긍정적인 영향을 미치는 갈등은 어느 정도 조장하고 부정적인 영향을 미치는 것은 제거해야 한다고 주장.
  - 갈등의 순기능
   = 선의의 경쟁을 유발, 촉진시킬 수 있음
   = 창의적, 쇄신적 행정의 원동력이 될 수 있음
   = 개인, 집단의 동태적인 성장과 발전의 계기가 될 수 있음
   = 행정의 획일성을 배제하고 다양성, 민주성을 확보하는 데 기여함
   = 조직의 새로운 조화와 통합을 이룰 수 있음
   = 조직발전과 쇄신의 계기가 될 수 있음
  - 역기능
   = 관계자의 심리적, 신체적 안정 동요, 인간관계의 안정적 균형 교란
   = 직원이나 기관 간의 적대 감정, 반목이 격화
   = 직원이나 기관 간 위계질서를 문란하게 하며 관계자의 사기 저하
   = 갈등이 격화되면 목표달성에 대한 관심이 줄어들고 행정의 조정이 어려워짐
   = 조직의 안정성, 생산성, 효과성, 적응성 저하

 * 욕구충족요인 이원론
  - 불만요인(위생요인)과 동기요인(만족요인)으로 나뉨
   = 위생/불만요인: 주로 환경에 관한 것으로, 직무에 불만족을 느끼게 하거나 예방하는 데 작용한다고 보면서 위생요인이라고 함
   = 동기/만족요인: 주로 직무 자체에 관한 것으로 인간에게 만족을 주고 우수한 직무수행을 위해 동기를 유발하는데 작용하는 요인, 즉 동기요인이라고 함
  - 한계
   = 이론을 개발할 당시의 연구대상이 기사와 회계사 등 전문직에 종사하는 사람이었기 때문에 다른 업종의 사람들에게는 일반화 어려움
   = 중요사건기록법을 근거로 수집되어 연구대상포본의 자아보호적 편경이 내포됨. 다른 조사방법을 쓰면 다른 결론이 도출될 수 있음
   = 개인차에 대한 고려 없으며, 여러 가지 위생요인이나 동기요인이 개인에게 미치는 영향은 개인의 연령이나 조직 내의 직위 등의 따라 다를 수 있음. 봉급 등 위생요인이 불만족을 방지하는 요인으로만 작용하는 것이 아니라 만족을 주는 요인으로도 작용할 수 있음
   = 직무요소와 동기 및 성과 간의 관계가 충분히 분석되어 있지 않고, 개인의 만족도와 동기수준의 관계에 대해서도 제대로 설명하지 못함.

 * 기대이론
  - 개요
   = 브룸의 기대이론은 동기부여의 과정적 측면을 중시함
   = 욕구 충족과 직무수행의 관계에서 직접적인 인과관계를 찾기 어렵다고 보아 욕구 충족과 동기 유발 사이에 인간의 주관적인 평과과정이 끼어든다고 보는 입장
   = 동기부여가 조직구성원이 보상에 대해서 얼마나 매력을 느끼고 있는가에 달림
   = 동기부여의 강도는 유의성(V), 수단성(I), 기대감(E)의 곱의 함수라고 함.
   = f[Σ유인가(V)x수단성(I)x기대감(E)]
  - 주요 내용
   = 기대감
    * 기대감은 특정 결과가 특정한 노력으로 인해 나타날 수 있다는 가능성에 대한 개인의 신념으로 주관적 확률로 표시
	* 브룸은 직무 수행 능력과 직무수행에 필요한 여러 가지 환경적 요인을 들음. 기대감은 완전한 의심인 0부터 완전한 믿음인 1 사이의 숫자
   = 수단성
    * 수단성은 1차 수준의 결과가 2차 수준의 결과를 가져오게 될 것이라는 개인의 믿음의 강도를 말함
	* 수치로 표현할 때 높은 성과가 낮은 성과를 가져올 것이라는 완전한 믿음인 -1에서, 성과와 보상 간에 아무런 효과가 없다고 믿는 0, 성과와 높은 보상간의 완전한 믿음인 1 사이로 나타낼 수 있음
   = 유의성
    * 특정 결과에 대해 개인이 갖는 선호의 강도를 말함. 원하는 특정한 보상에 대한 선호의 강도를 유의성
	* 유의은 보상에 대해 그 개인이 느끼는 보상의 중요성 의미. 보상을 받지 않았을 때보다 받았을 때 선호를 느끼면 정의 유의성, 그 반대에는 부의 유의성 가짐
   = 결과
    * 개인행동의 성과와 같은 1차적 결과와 성과에 따른 승진 등과 같은 2차적 결과, 즉 보상으로 구분함
	* 높은 수준의 성과는 봉급인상, 승진, 상사로부터 받는 인정, 피로, 스트레스, 휴식시간의 부족 등을 초래함
	* 봉급인상, 승진, 상사의 인정 등은 일반적으로 정의 유의성. 피로, 스트레스, 휴식시간의 부족 등은 일반적으로 부의 유릐성을 가짐
	* M = f[Σ(A->P1)(P1->P2)xV]

5> 조직의 혁신
 * TQM(총체적 품질관리)
 - 개념
  = 총체적 품질관리는 고객에 대한 서비스의 질을 높이기 위한 관리(세 가지 경영철학: 고객만족, 전사적-전직원의 참여, 지속적 개선)기법으로 고객만족 중시하는 관리철학임
  = 공공서비스 품질 향상을 통한 고객만족을 목표로 하기 때문에 공무원들의 행태를 고객 중심적으로 전환 가능
  = 지속적 업무 개선, 전체 구성원의 참여, 권한위임 등의 조직문화 형성
  = 민간부문에서 처음 이루어졌고, 1980년대 미 국방부와 농무성에서 처음 시행
 - 코헨과 브랜드의 정의
  = Total: 업무의 모든 면에 대한 조사를 적용하는 것
  = Quality: 고객의 기대수준을 충족시킬 뿐 아니라 그것을 초과
  = Management: 업무의 질을 향상시키기 위하여 조직의 능력 개발하는 것
 - 특징
  = 품질은 고객에 의해 정의, 재화와 용역을 생산하는 과정을 중시하여 인간 위주의 경영시스템 지향. 고객만족을 달성하여 궁극적으로 조직의 장기적 성장 추구
  = 고객만족을 최종목표로 하기 때문에 공무원들의 행태를 고객 중심으로 전환할 수 있음
  = 계층제조직과 비교해 보았을 때 탄력석을 그 특징으로 하며 구성원 전체의 참여와 권한 및 책임 부여를 통해 조직 내 인간관계를 원활하게 하고 사기를 진작하게 함
 - 대두 배경
  = 미국 통계학자 데밍에 의해 개발되어 일본기업에 성공적으로 적용됨
  = 행정개혁의 해결책을 정부 내부에서 구하는 것, 주 공급자로서의 지위를 상실하지 않으면서도 민영화와 같은 시장 확대전략에 대처하기 위해 나온 것으로 해석 가능
 - 기본 원칙
  = 고객이 품질의 최종 결정자
  = 품질은 전 구성원에 의해 결정되고 구성원 참여가 강조
  = 산출과 결과보다는 투입과 과정의 계속적 개선 요청
  = 전체적인 조직의 책임 강조(조직의 총체적 헌신)
  = 산출과정 초기에 품질 정착
  = 서비스의 지나친 변이성 방치
  = 소비자 중심의 품질
  = 명확하고 공유되는 목표
  = 참여적 관리(Y이론적 인간관)

 * BSC(균형성과관리)
 - BSC의 개요
  = 균형성과괸리는 하버드 비즈니스 스쿨의 로버트 캐플런과 경영컨설턴트인 네이비드 노턴이 재무지표 중심의 기존 성과관리의 한계를 극복. 즉 BSC는 비전과 전략 달성을 위해 수행해야 할 핵심적인 사항을 측정 가능한 형태로 바꾼 성과지표의 집합이며, 전통적인 재무재표뿐만 아니라 고객, 비즈니스 프로세스, 학습, 성장과 같은 비재무인 측면도 균형적으로 고려함
  = 고객 중심 경영 및 장기적 성장가능성의 개념이 부합되는 것이라 할 수 있음. 이러한 지표를 통해 전략 수행을 위한 핵심적인 영역을 조직원에게 명확히 전달할 수 있게 되고, 이를 관리함으로써 조직의 전략수행 여부를 모니터할 수 있게 함
 - BSC의 핵심 지표
(1) 재무적 관점
  = 재무적 지표와 비재무적 지표, 조직의 내부 요소와 외부 요소, 결과를 예측해 주는 선행지표와 결과인 후행지표, 단기적 관점과 장기적 관점의 균형 중시
  = 재무지표를 의미하는 것으로 전통적인 후행지표
(2) 고객 관점
  = 전달해야 하는 가치를 고려해야 하는 것을 말하는 것으로, 공공부문에서 BSC를 도입할때 가장 중요하게 고려해야 하는 점
(3) 내부 프로세스 관점
  = 효과성을 극대화하여 남보다 뛰어난 경쟁령 확보, 고객의 요구를 가장 신속하게 파악하고 충족시켜 주는 내부 프로세스 보유함
(4) 학습과 성장 관점
  = 장기적 관점으로 역량, 지식의 축적, 정보시스템 구축 등과 관련
  = 지식가치의 극대화가 목표이며 직무만족도가 높은 경우에 가능함
 - BSC의 기능
(1) 성과측정시스템으로서 BSC
  = 재무적 관점, 고객 관점, 내부 프로세스 관점, 학습과 성장 관점이 균형을 이루면서 조직의 목표달성 전략 구현하기 위해 개발된 핵심 성과지표 간 연계 체계
(2) 전략관리시스템으로서의 BSC
  = 4개의 관점별로 전략목표, 성과지표, 목표값과 실행계획으로 전환해 완성되는 전략관리시스템임. 특히 전략목표가 개인에까지 연결되는 캐스케이딩 과정은 전 조직을 전략 집중형 조직으로 변화시키는 기능을 수행함
  = 목표값과 실행계획은 전략과 이를 구형하기 위한 조직의 예산과 인적 자원 배분을 연계하는 기회 제공
(3) 의사소통 도구로서 BSC
  = 조직구성원들에게 조직의 전략 목적을 달성하기 위해 필요한 성과가 무엇인지를 알려 주기 때문에 조직전략의 해석지침이 됨
  = 성과지표의 방향을 따라가면 추구하는 비전과 미션, 가치 발견함
  = 구성원에게 전하고 싶은 메세지가 성과지표의 형태로 전다
 - BSC의 장점
  = 각 입장을 치우침 없이 반영
  = 기업 -> 부문 -> 개인의 순으로 낮춤으로써 조직의 말단에까지 비전 실현 의지 높임
  = IT를 도입함으로써 내비게이터 역할 수행. 빠른 문제 발견과 대처 가능
  = 피드백을 컨트롤, 피드포워드(징후 계산에 의한 예측과 제어) 컨트롤이 가능
  = 기본적인 수치와 목표 설정함으로 전략과 비전의 도달 정도를 실감하고 확인할 수 있음
