02. 정책학
1> 정책학 기초이론
 * 정책유형
  - 로위: 배분정책, 규제정책, 재분배정책, 구성정책
  - 리플리&프랭클린: 배분정책, 경쟁적 규제정책, 보호적 규제정책, 재분배성정책
  - 알몬드&파웰: 배분정책, 규제정책, 추출정책, 상징정책
  - 세일스버리: 배분정책, 규제정책, 재분배정책, 자율규제정책

 * 정책결정요인론
  - 개념
   = 정책의 내용을 결정하는 요인이 무엇인가를 밝히려는 노력
   = 사회경제적 요인이 정책의 내용을 좌우한다고 주장함.
  - 경제학자들의 연구
   = 정치학자의 연구 결과 정치적 변수가 정책에 의미 있는 결과를 미치지 못함
   = 민국 민주주의 기본 이념인 정치적 다원론에 큰 타격
  - 정치학자들의 재연구
   = 정치체제와 사회경제적 변수를 모두 포함하는 새로운 이론의 구성을 시도함
   = 그동안 이론의 전제인 사회경제적 변수, 정치체제, 정책 간 순차적 관계 부정. 모든 변수가 정치체제와 정책의 상관관계를 초래함
   = 사회경제적 변수를 통제하면 정치체제와 정책의 관계는 사라지는데, 이는 양자의 관계가 허위상관에 불과하다는 반증
 * 하위정부론
  - 개념
   = 비공식 참여자로 분류되는 이익집단과 공식적 참여자인 관료조직, 그리고 의회의 위원회 간의 연계적인 활동을 통한 정책의 결정과 결과에 주목하는 모형
   = 이익집단, 의회의 위원회, 해당 관료조직의 3자 연합(철의 삼각형)이 각 정책영역별 정책의 결정과 집행에 영향을 미친다고 봄
   = 하위정부는 모든 정책 분야에 걸쳐서 가능한 것이 아니라, 대통령의 영향이 적은 배분정책에서 주로 형성됨
  - 장점: 외부로부터의 개입을 배제하면서 상당히 독립적인 역할을 수행. 따라서 하위정부모형은 구성원 간의 관계가 안정적이며, 정책자율성이 높은 정책네트워크의 유형
  - 단점
   = 정책과정에서 이익집단의 역할이나 활동을 지나치게 강조하고 있음. 분야별 이익집단이 활성화되어 있지 못한 정치체제하의 정책 과정을 설명하는데 한계
   = 분야별 이익집단이 활성화되어 있는 정치체제의 경우에 정책문제가 많은 사람들의 관심을 끌고 논쟁적인 성격을 지닌 경우에 이익집단이 자기들의 이익을 관철하기 위하여 주도적인 역할을 수행하는 것이 여의치 않을 것이라는 점 간과
  
 * 정책네트워크 
  - 개념
   = 네트워크 분석을 다양한 참여자들의 행위로 특정지어지는 정책과정의 연구에 적용하는 것으로, 기본적으로 행위자들 간의 관계를 중시함
   = 네트워크 분석에서 행위자들 간의 관계는 자원의존성을 토대로 한 교환관계를 중심으로 하고 있움. 여기서 행위자들 간의 관계 밀도와 중심성의 개념을 중심으로 점, 선, 파라미터 등으로 네트워크 표현
 * 조합주의
  - 개념
   = 1920~1930년대 유럽의 파시스트 조합주의가 기업가들을 일방적으로 이롭게 할 목적으로 파시스트 당원들을 대표로 받아들인 후, 노동자 대표를 기업가 대표와 강제유착시킨 것
   = 사회적인 통합을 지나치게 강조하여 자본주의 요소인 경쟁과 갈등의 존재를 무시함. 제 2차 세계대전 후 파시스트 국가가 패망하면서 잊힌 정치 용어가 됨
  - 국가조합주의
   = 국가가 일방적으로 위로부터 제도적 장치를 강제 부과하는 것
   = 파시스트 체제, 제3세계, 후진 자본주의 사회의 권위주의적 정체제제의 체제의 특징을 지님(후진국)
  - 사회조합주의
   = 사회경제체제의 변화에 순응하려는 이익집단의 자발적 시도로부터 생성된 것
   = 의회 민주주의를 통하여 발전되고 있는 이익대표 체제의 특성 지님(선진국)
  - 신조합주의: 다국적 기업과 같은 중요 사업조직이 국가 또는 정부와 긴밀한 동맹관계를 형성하고 이들이 경제 및 산업정책을 함께 만들어 간다는 이론
 
2> 정책의제설정론
 * 정책의제설정과정
  - 콥과 엘더의 모형
   = 사회문제 -> 사회적 이슈 -> 공중의제 -> 정부의제
  - 콥과 로스의 모형
   = 문제제기 -> 구체화 -> 확장 -> 진입
  - 아이스톤의 모형
   = 문제태동 -> 집단인지 -> 이의를 가진 집단의 관여 -> 사회적 이슈의 표출 ->  공중의제 형성 -> 쟁점참여자의 활동 -> 공식의제 채택 -> 정책결정집단에 의한 관련 쟁점 표출
  - 존스의 모형
   = 사건 인지와 문제의 정의 -> 결속화 및 조직화 -> 대표 -> 의제 체택
  - 켄드릭의 모형
   = 문제진술과정 -> 문제전달과정 -> 조직화과정
 * 정책의제설정의 모형
  - 외부주도모형
   = 개념: 정책담당자가 아닌 외부사람들의 주도에 의한 것
   = 특징: 이익집단이 발달하고 정부가 외부의 요구에 민감하게 반응하는 정치체제에서 주로 나타나기 때문에 다원화된 선진국의 정책의제설정과정에서 나타나며, 다원화된 선진국의 정책의제설정과정에서 주로 나타남.
   = 예시: 전자상거래제도, 벤처산업육성, 개방형 임용제, 소비자보호정책 등
  - 동원모형
   = 개념: 정책결정자들이 주도하여 정책의제를 채택하는 경우로, 정부가 민간을 동원하여 의제를 설정하는 것
   = 특징: 정부가 힘이 강하고 민간부문의 힘이 취약한 후진국에서 많이 나타나는 유형. 정부가 미리 문제를 해결할 것을 결정한 다음, 일반대중에게 이해시키기 위한 PR 활동을 진행
   = 예시: 새마을운동, 올림픽 유치 등
  - 내부접근모형
   = 개념: 관료집단이나 외부집단에 의하여 주도되어 이들이 최고 정책결정자에게 접근하여 정책의제로 채택되는 경우를 말함
   = 특징: 자발적으로 정책의제화가 진행되지만 외부 국민들과는 상관없이 정부관료제 내부에서 정책의제화가 이루어지며 공중의제는 성립되지 않음
   = 예: 국민에게 알려지기 곤란한 것
 * 정책의제설정 영향 요인
  - 정책문제의 성격
  - 정책담당체제의 특성
  - 문제 인지자의 역량
  - 정치상황
  - 경제, 사회상황
  - 정책 이해관계자의 특성
 
 * 무의사결정
  - 배경: 바흐라흐와 바라츠가 다알의 연구를 비판한 데서 비롯함. 정치권력은 두 가지 얼굴을 가지고 있는데, 하나는 정책문제 해결을 위한 권력이고, 다른 하나는 갈등이 정체과정에 진입하지 못하게끔 방지하는데 사용됨
  - 개념: 사회문제 중 일부만이 정책의제화되고 나머지는 기각되는데, 이는 소극적 의사결정의 일부이므로 무의사결정이라고 함. 
  - 발생 원인과 방법
   = 원인: 지배계급에 대한 이해관계의 침해, 지배계급의 가치관에 대한 도전, 문제해결이 저지되도록 정치체제가 구조화된 경우, 특정 개인 및 집단의 이익에 편파되는 편견의 존재, 과잉충성
   = 방법: 폭력, 권력의 행사, 특혜의 부여, 편견의 동원, 강화, 수정, 문제 자체의 은폐 및 지연전략

3> 합리적 의사정책 결정과 정책분석론
 * 문제정의
  - 정책문제의 구성요소, 원인-결과 등의 내용을 규정하여 무엇이 문제인지를 밝히는 과정
  - 관련 요소 파악: 정책문제 유발 인물과 사물의 존재, 상황 요소를 찾아내는 작업
  - 가치판단: 원하는 가치가 무엇인가를 판단하고 가치들 사이의 보완, 경쟁관계, 주종 관계 파악
  - 인과관계 파악: 관련 요소들의 관계를 원인, 매개, 결과로 나누어 보고 이들 변수를 관계식으로 연결시켜 이해하는 것
  - 역사적 맥락 파악: 요소들 사이 관계를 단순히 정태적으로 파악할 것이 아니라, 각 요소들의 역사적 발전과정, 변수들 사이의 관계 변화과정에 대해서도 살펴보아야함
 * 집단사고
  - 응집력이 강하고 적은 인원 수로 구성된 정책결정 집단일수록 집단 사고방식으로 인한 합리적 정책결정을 벗어나는 경우가 많음. 구성원 간 만장일치를 위해 이의가 묵살되는 형태가 나타남.
 
 * 델파이분석
  - 개념 및 배경
   = 전통적인 회의식 기법이 갈등이 심하고 분위기에 영향을 받게 된다는 비판에서 등장
   = 델파이기법은 예측하려는 현상에 대해 관련 있는 전문가의 자문을 설문지를 통해 근접한 의견에 이를 때까지 체계적으로 유도하고 분석하는 직관적인 미래예측기법을 말함.
   = 문제가 복잡하고 경험적 자료가 없으며 이론도 존재하지 않는 경우 사용될 수 있음. 중장기적 미래 예측
  - 특징
   = 익명성
   = 반복
   = 통제된 환류
   = 통계적 처리
   = 합의
  - 장점
   = 외부적인 영향력으로 결론 왜곡 방지
   = 통제된 환류과정을 반복함으로써 주제에 대한 관심 제고할 수 있음
   = 응답의 결과가 통계적으로 처리됨으로써 비교적 객관적인 결론을 도출 가능 (그러나 미래예측의 근거는 전문가의 주관과 직관)
   = 전문가 집단을 동원하게 되므로 예측 분야와 관련되는 많은 변수를 고려할 수 있음
   = 다루어질 정보의 양이 증가될 수 있음
   = 집단적 상호작용을 통하여 보다 많은 지식과 자극을 나눌 수 있음
   = 미래예측에 있어서 위험 부담이 적음
  - 단점
   = 동원된 전문가들 자질과 역량이 문제될 수 있음
   = 델파이과정에서 응답이 불성실하거나 조작가능성 있음
   = 설문 여하에 따라 응답이 크게 달라질 수 있음
   = 예측된 사건들 사이의 잠재적인 관계가 무시될 수 있음
   = 예측이 상호보강적이거나 상호배타적인 항목을 포함하는 경우 많음
   = 그릇된 정보를 더 많이 이용하기 쉬움
   = 소수 의견이 묵살될 가능성이 많음
   = 전문가로서 체면을 세우기 위하여 의견 고집
   = 이해관계가 작용하는 정치적 의사결정의 가능성 있음
 * 정책델파이
  - 개념: 정반대의 입장에 있는 서로 다른 대립되는 의견을 표출시키는 목적.
  - 특징
   = 선택적 익명성
   = 양식 있는 많은 주창
   = 차이를 부각시키는 통계 처리
   = 조성된 갈등: 각종 의견 기저의 가정이나 진술을 표출하여 갈등을 활용. 합의되지 않고 계속 갈등형식으로 남을 수 있음
   = 컴퓨터를 통한 회의방식

 * 비용편익분석
  - 개념
   = 투입되는 비용과 산출량의 상관관계를 고려햐여 편익이 큰 것을 기준으로 예산안 선택 또는 우선순위를 명백히 하는 것을 의미. 편익은 금액화할 수 있음
   = 모든 편익과 비용 열거 -> 화폐액으로 환산 -> 할인율을 적용하여 현재가치로 계산 -> 우선순위 선정
  - 비용과 편익의 추계
   = 비용의 추계: 비용은 미래에 발생할 비용만 계상되며, 기회비용은 고려하나 매몰비용은 고려하지 않음
   = 편익의 추계: 미래에 발생할 편익을 현재의 화폐가치로 환산해서 평가함. 무형적인 편익과 유형적 편익, 파급효과를 고려해야 함. 실질적인 편익은 고려하나, 금전적인 편익은 사회 전체적으로는 순이득이 아니므로 비용편익시 제외된다. (이 경우 편익은 화폐화되지 않음)
 
 * 정책대안의 평가기준 및 방법
  - 순현재가치(NPV)
   = 순현재가치 = 편익의 총현재가치 - 비용의 총현재가치
   = 한계순현재가치 = D편익의 현재가치 - D비용의 현재가치
  - 편익비용비율(B/C Ratio)
   = 편익비용비율 = 편익의 총현재가치 ÷ 비용의 총현재가치
   = 한계편익비용비율 = D편익의 총현재가치 ÷ D비용의 현재가치
  - 내부수익률(IRR)
   = 'NPV = 0' 또는 'B/C Ratio = 1'이 되도록 하는 할인율
   = '한계순현재가치 = 0'이 되도록 하는 할인율
   = 복수의 대안편가 시, 내부수익률이 큰 사업을 선택하게 된다.
 * 불확실성 대처방안
 

4> 의사정책결정 이론모형
 * 정책결정모형
  - 개인적 의사결정모형: 합리모형, 만족모형, 점증모형, 혼합모형, 최적모형
  - 집단적 의사결정모형: 회사모형, 쓰레기통모형, 앨리슨모형

 * 합리모형
  - 결정기준: 경제적 합리성, 인간의 전지전능성
  - 학자: 경제학자
  - 한계: 이상적, 비현실적, 인간의 한계
  - 집단모형: 공공선택모형, 엘리슨 1모형
 * 만족모형
  - 결정기준: 제한된 합리성, 심리적 만족도
  - 학자: 사이먼&마치
  - 한계: 주관적, 보수적, 모형의 일반화가 곤란
  - 집단모형: 연합모형, 회사모형, 앨리슨 2모형
 * 점증모형
  - 결정기준: 제한된+정치적 합리성, 현재보다 나은 수준
  - 학자: 린드블룸&윌다브스키
  - 한계: 보수적, 선진국형 개도국 적용 곤란
  - 집단모형: 쓰레기통모형, 앨리슨 3모형
 * 사이버네틱스모형
  - 결정기준: 제어기능의 조절과 통제를 통한 지속적인 적응에 초점
  - 한계: SOP를 통한 순응적인 의사결정
 * 혼합모형
  - 결정기준: 기본-합리모형-숲, 부분-점증모형-나무
  - 학자: 에치오니
  - 한계: 혼합에 불과
 * 최적모형
  - 결정기준: 합리모형 + 초합리성, 환류
  - 학자: 드로어
  - 한계: 합리모형의 한계 답습, 초합리성의 개념 모호
 
 * 회사모형
  - 배경
   = 사기업의 사장은 유기체의 두뇌와 같고 밑의 사람은 손발과 같이 움직인다는 것과, 완전경쟁시장하에서 기업의 구성원은 경제인과 같다는 가정을 함
  - 개념
   = 사이어트와 마치가 개인적 의사결정에 치중한 만족모형을 발전. 연합모형이라고 함
   = 상호제약된 독립적 합리성을 추구하는 하위 조직간에는 목표가 서로 대립함
   = 조직 하위 간의 갈등의 해결이 곧 조직의 의사결정이라고 보며, 갈등은 협상을 통해 불완전 해결된다 봄
  - 특징
   = 갈등의 준해결
   = 불확실성의 회피
   = 문제 중심적 탐색
   = 조직체의 학습
   = 표준운영절차의 중시
   = 상호협상을 통한 의사결정
  - 한계
   = 공공부문 적용에 한계
   = SOP를 따르는 결정방식은 급격한 변동상황에 부적합하여 경직성 초래. 개도국의 경우에는 적응이 힘듬
   = 광범위한 권한위임 및 자율성이 강한 조직을 전제함으로 권위주의 조직의 적용에 한계가 있음
 * 쓰레기통모형
  - 개요
   = 복잡하고 변화 및 혼란한 상황에서 조직의 결정 형태에 초점을 둔 이론모형
   = 혼란한 상황에서 설득력과 설명력을 지니지만 가장 비합리적인 모형임
   = 조직화된 무정부 조직에서 나타나는 몇 가지 흐름에 의하며 정책결정이 우연히 이루어진다고 보는 정책모형
  - 내용
   = 올슨, 코헨, 마치 등이 주장
   = 쓰레기통에 쓰레기들이 모여 있듯이 우연하게 의사결정이 결정됨
   = 시행착오에 의존하거나 과거의 경험이나 실용적 발견 또는 필요성에서 우연히 얻은 교훈에 의하여 정책이 결정됨
  - 전제조건
   = 문제성 있는 선호
   = 불명확한 기술
   = 일시적 참여자
  - 의사결정의 방식
   = 진빼기 결정, 날치기 통과
  - 비판
   = 실제 조직은 쓰레기통모형보다 훨씬 더 체계적임
   = 무질서한 정책결정 요소들은 실제로 반영되지 않음

 * 앨리슨 모형
  - 모형 1: 합리모형
   = 한 나라의 정부를 단일적, 합리적인 행동주체로서 파악하고 정부의 의사결정을 분석함
   = 통합된 목표를 가지며 이 목표의 달성을 위해 모든 수단을 강구하고 적절한 것을 선택함
  - 모형 2: 조직과정모형
   = 정부의 목표에 부합되는 선택보다 정부를 구성하는 다양한 SOP에 따르는 전형적 행동의 표출로 봄
   = 정부는 느슨하게 연결된 반독립적인 하위 조직의 집합체로 간주함.
   = 문제가 발생하면 SOP나 기존 프로그램에 의존함
  - 모형 3: 관료정치모형
   = 의사결정이 참여자 간의 타협, 흥정으로 이루어지는 정치활동으로 봄
   = 서로 다른 인지구조, 문제에 대한 상이한 인식, 해석, 옹호할 목표 또는 대안에 차이가 있음을 전제
   = 문제 자체에 대한 해석, 목표설정과 수단 선택에 관하여 논의, 설득, 흥정이 진행됨
  - 평가
   = 우리나라에서는 대통령의 권한이 막강해서 적용이 힘들었으나 민주화의 결과로 그 적용가능성이 점차 높아짐
 * 자명반론자기법
 
5> 정책집행론
 * 고전적, 하향적, 집권적 집행 (탑-다운)
  - 정책상황: 안정적, 구조적 상황
  - 정책목표의 수정: 목표가 명확하여 수정 필요성 맞음
  - 관리자의 참여: 참여 제한, 충실한 집행 중시
  - 집행자의 재량: 집행자의 재량 불안정
  - 정책평가의 기준: 집행의 충실성과 정책 결과
  - 결정과 집행: 정책집행은 결정과 독립적 과정(정치-행정 이원론)
  - Berman: 정형적 집행
  - Elmore: 전향적 집행
  - Nakamura: 고전적 기술자형, 지시적 위임형
  - 일선관료제: 무관심
 * 현대적, 상향적, 분권적 집행 (바텀-업)
  - 정책상황: 유동적, 동채적 상황
  - 정책목표의 수정: 수정 필요성 높음
  - 관리자의 참여: 참여 필요
  - 집행자의 재량: 집행자의 재량 인정
  - 정책평가의 기준: 환경에 대한 적응성 중시, 정책 성과는 2차적 기준
  - 결정과 집행: 정책집행은 결정과 유기적 과정 (정치-행정 일원론)
  - Berman: 적응적 집행
  - Elmore: 후향적 집행
  - Nakamura: 재량적 실험가형, 관료적 기업가형
  - 일선관료제: 관심(재량권 확대)

 * 일선관료제
  - 개념
   = 립스비는 업무수행과정에서 시민과 직접적으로 접촉하며 업무수행상 상당한 재량을 보유하는 공무원을 일선관료라 하며, 그 구성원의 상당 부분에 일선관료로서 구성되는 공공서비스 기관을 일선관료제라고 규정함
   = 사회복지요원, 경찰, 교사 등ㅇ이 일선관료에 포함
  - 일선관료제의 특징
   = 대면 처리적 업무가 주임
   = 고객을 범주화하여 선별
   = 많은 재량을 행사한다
   = 업무과다와 자원의 부족에 직면함
  - 일선관료의 업무환경
   = 문제성 있는 업무환경을 다음과 같이 정리
   =자원의 부족 문제: 시간과 정보의 부족, 기술적인 지원의 부족 등
   = 권위에 대한 도전: 육체적, 정신적 위협이 큰 환경에 처한 경우 자신들의 권위를 보장받으려는 경향 커짐
   =모호하고 대립되는 기대의 존재: 비현실적이고 상호갈등을 발생시키는 경우 많음. 이런 경우 일선관료는 그 중 하나의 기대만을 선택하게 되는 경향
  - 고정관념의 경향
   = 단순화와 정형화를 통해 복잡한 문제와 불확실한 상황에 대처함. 단순화, 정형화와 같은 적응 메커니즘을 형성하는데 가장 큰 영향을 미치는 것은  일선관료들의 고정관념
   = 자원이 부족할 경우 부분적 또는 간헐적 집행
   = 위협을 공개적으로 과장시켜 공식적인 징계 경감
   = 역할에 대한 기대를 변경, 고객집단을 재정의하거나 고객이나 사회에 책임을 전가시켜 상호모순되거나 모호한 역할기대를 회피한다.
 
 * 정책지지가연합모형
  -대표적인 통합모형
  - 상향식 접근방법의 분석 단위 채택. 하향식 접근방법의 여러 가지 변수를 결합. 상향식 접근방법에 의해 정책문제나 일선 조직에서 검토를 시작하여 공공부문과 민간부문에서 행위자들의 전략적 행위에 초점. 하향식 접근방법의 관점에서 법적, 사회경제적 변수의 영향 분석함
  - 행위자의 집단 을 구분하는 기준으로 신념체계를 사용. 행위자 집단인 지지연합의 정책학습 강조. 신념체계별로 여러 개의 연합으로 구성된 정책행위자 집단이 자신들의 신념을 정책으로 관찰하기 위해 경쟁한다는 점을 강조. 정책문제가 쟁점에 적극적으로 관심을 가지는 공공 및 민간 조직의 행위자들로 구성되는 정책하위체계 개념을 사용함
 
 * 나마쿠라와 스몰우드의 모형
  - 고전적 기술자형
   = 정책결정자: 명백한 목표 제시
   = 정책집행자: 목표달성을 위한 기술적 권한만 위임
   = 평가기준: 정책목표의 달성도(효과성)
  - 지시적 위임가형
   = 정책결정자: 명확한 목표 제시
   = 정책집행자: 목표달성을 위한 기술적 권한 + 행정적, 관리적 권한 위임
   = 평가기준: 효과 또는 능률성
  - 협상자형
   = 정책결정자와 정책집행자: 결정자와 집행자 간에 정첵목표와 정책수단에 대해 협상
   = 평가기준: 정책지지 및 유권자의 만족도
  - 재량적 실험가형
   = 정책결정자: 경절자의 지식 부족 등으로 인해 추상적, 일반적 목표만 설정
   = 정책집행자: 목표 구체화의 수단의 개발에 관한 광범위한 재량을 위임
   = 평가기준: 정책수혜집단의 대응도
  - 관료적 기업가형
   = 정책결정자: 결정자와 수단에 대해 협상
   = 정책집행자: 집행자가 정책과정 전반을 완전히 통제
   = 평가기준: 체제 유지
 
 * 행정수단 - 직접성 정도에 의한 분류(살라몬)
  - 낮은 직접성
   = 정책수단: 손해책임법, 보조금, 대출보증, 정부출자기업, 바우처
   = 효과성: 낮음
   = 효율성: 높음
   = 형평성: 중간
   = 관리 가능성: 낮음
   = 정당성(정치적 지지): 높음
  - 중간 직접성
   = 정책수단: 조세지출, 계약, 사회적 규제, 벌금
   = 효과성: 낮음/중간
   = 효율성: 중간
   = 형평성: 낮음
   = 관리 가능성: 낮음
   = 정당성(정치적 지지): 높음
  - 높은 직접성
   = 정책수단: 버함, 직접 대출, 경제적 규제, 정보 제공, 공기업, 정부 소비
   = 효과성: 높음
   = 효율성: 중간
   = 형평성: 높음
   = 관리 가능성: 높음
   = 정당성(정치적 지지): 낮음

 * 행정수단 - 강제성 정도에 의한 분류
  - 낮은 강제성
   = 정책수단: 손해책임법, 정보 제공, 조세 지출
   = 효과성: 낮음
   = 효율성: 중간
   = 형평성: 낮음
   = 관리 가능성: 중간
   = 정당성(정치적 지지): 높음
  - 중간 강제성
   = 정책수단: 바우처, 보험, 보조금, 공기업, 대출보증, 직접 대출, 계약 벌금
   = 효과성: 중간
   = 효율성: 높음
   = 형평성: 중간
   = 관리 가능성: 중간
   = 정당성(정치적 지지): 중간
  - 높은 강제성
   = 정책수단: 경제적 규제, 사회적 규제
   = 효과성: 높음
   = 효율성: 높음/낮음
   = 형평성: 높음
   = 관리 가능성: 낮음
   = 정당성(정치적 지지): 높음/낮음

 * 행정수단 - 일반적(정부개입의 정도에 의한) 분류
  - 강제적 수단: 정부의 직접 시행, 법과 규제, 공기업, 서비스 공급 등
  - 혼합적 수단: 보조금, 조세감면, 지급보증, 기타 경제적 수단
  - 자발적 수단: 민간부문(시민단체, 시장경제 등)의 자율적 활동

6> 정책평가론
 * 정책평가의 목적
  - 정책이 국민의 요망에 어느 정도 대응하고 있는가, 즉 정책에 대한 국민의 만족도를 파아갛ㄹ 수 잇게 함.
  - 정책평가의 결과 정보는 합리적인 정책의 결정,  수정, 보완, 종결 등에 도움이 됨
  - 합리적인 자원의 재배분을 가능하게 함

 * 총괄평가
  - 정책이 집행된 후 그 정책이 의도했던 효과를 초래했는가를 평가하는 것으로, 수단과 목표 사이에 존재하는 인과관계를 검증하는 평가임. 총괄평가에는 효과성평가, 능률성평가, 형평성평가가 있음. 양적 방법에 의존함
 * 형성평가
  - 집행과정에서 나온 여러 문제점을 해결하여 좀 더 나은 집행전략과 방법을 구하고자 하는 평가임. 질적 방법에 의존함 

 * 내적 타당성 저해요인
  - 역사적 요소
  - 통계적 회귀요소
  - 성숙효과
  - 피실험자 상실
  - 실험효과
  - 측정도구 변화
  - 선발과 성숙의 상호작용
  - 처치와 상실의 상호작용
 * 외적 타당성 저해요인
  - 실험조작의 반응효과(호손효과)
  - 크리밍효과
  - 실험조작과 측정의 상호작용
  - 표본과 대표성 부족
  - 다수적 처리에 의한 간섭
  
 * 정책평가의 논리와 방법
  - 비실험적 방법
   = 사후적 비교집단 구성방법
   = 정책 실시 전후 비교설계방법
   = 문제점
  - 실험적 방법
   = 진실험
   = 준실험
   = 비교: 내적 타당성에서는 진실험이, 외적 타당성과 실행가능성에서는 준실험이 우수함
 * 준실험설계 
  - 비동질적 통제집단 설계에 의한 평가, 사후측정 비교집단 설계, 회귀불연속 설계, 단절적 시계열설계 등이 있음
