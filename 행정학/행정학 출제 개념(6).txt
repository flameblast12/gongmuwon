06. 행정 환류론
1> 행정책임과 통제
 * 행정통제의 유형
  - 법적 책임과 재량적 책임
  - 도의적 책임과 법적 책임
  - 내재적 책임과 외재적 책임
  - 정치적 책임과 기능적 책임
  - 제도적 책임성과 자율적 책임성
  - 시장책임성

 * 행정통제의 과정
  - 통제목표와 기준의 설정
  - 성과측정 및 기준과의 비교, 평가
  - 시정조치
 
 * 내부통제
  - 공식적 통제
   = 청와대와 국무총리실에 의한 통제
   = 감사원에 의한 통제
   = 중앙통제
  - 비공식적 통제
   = 행정인의 직업윤리에 의한 자율통제 의미
   = 행정의 전문화 심화와 재량권의 증대로 인해 현대에 이르러 가장 효율적, 합리적 통제로서 중요시됨
 * 감사원
  - 행정통제와 관련된 감사원의 기능은 직무감찰과 회계검사로 구분됨
  - 최근에는 추가적으로 성과감사의 강화가 요청되고 있음
  - 즉 기본적으로 수행하던 감사와 더불어, 행정부가 추진하는 각종 정책에 대한 성과 중심의 감사를 강화해야 한다는 것.

 * 옴부즈만(호민관, 행정감찰관)
  - 개념: 옴부즈만이란 행정이 합법적, 합목적적으로 수행되고 있는가를 시민들이 권리구제 신청에 의하거나 직권에 의하여 조사, 감찰하는 기관
  - 발달
   = 전통적인 행정통제 방식인 입법부, 사법부에 의한 통제가 약화됨에 따라 입법부에 의한 행정부 통제수단으로서 필요하게 됨
   = 1809년 스웨덴에서 최초로 채택된 이래 핀란드, 노르웨이, 덴마크, 뉴질랜드, 영국, 미국, 이스라엘, 캐나다 등에서 채택하여 시행됨
  - 특징
   = 입법부 소속 공무원이며, 입법부에서 선출됨(외부통제)
   = 직무수행의 독립성과 임기 보장
   = 당파성이나 정치성이 없는 조사직
   = 행정결정을 취소 또는 변경할 수 있는 권한 없음. 법원, 행정기관에 대한 직접적인 감독권이나 통제권은 없음. 옴부즈만을 이빨 없는 집 지키는 개라고 함
   = 시민의 권리구제 신청이 없어서도 직권조사 권한을 가짐
   = 헌법기관이며, 합법성뿐만 아니라 합목적성의 문제에 관해서도 조사, 처리할 수 있음
   = 권한으로 독립적 조사권, 시찰권, 소추권 등이 있는데, 소추권은 인정하지 않는 것이 보통
  - 장점
   = 법원의 경우와는 달리 신속히 처리되며 비용 저렴
   = 시민의 접근 용이
   = 제도 수립에서 고도의 융통성과 적응성 지님
   = 특히 대민 행정량과 인구가 적은 사회에서 큰 효용
   = 행정운영의 개선과 정치발전의 계기를 마련할 수 있음
   = 국민의 피해의식 감소, 시민의 참여의식 향상
  - 단점
   = 국회 및 기타 기존의 타기관, 타제도와 기능상의 중복 초래
   = 직접 통제, 감독권 없음
  

2> 행정개혁(정부혁신)

 * 구조적 접근방법
  - 개념: 조직의 구조적 설계를 개선함으로써 행정개혁의 목적을 달성하는 접근방법
  - 전략
   = 원리전략: 주로 전통적 조직이론에 근거를 두며, 조직의 건전원칙에 의거한 최적조구가 업무의 최적수행을 가능하게 함
   = 분권화 전략: 분권화 전략은 구조의 분권화에 의해 조직을 개선하려는 것
  - 예시
   = 분권화의 확대
   = 통솔범위의 조정
   = 통솔범위의 조정
   = 의사결정 권한의 수정
   = 행정조직의 계층 간 의사전달 체계의 수정 등을 통한 조직의 개선

 * 관리기술적 접근방법
  - 개념: 업무수행과정에서 능률을 향상시키기 위해 새로운 행정기술 장비를 도입하거나 관리과학, 운영연구, 체제분석, 컴퓨터 등의 계량화 기법을 활용하는 것
  - 장점: 기술적 쇄신을 통해 표준적 절차와 조직의 과업수행에 영향 줌, 조직행태와 인간행태에 영향을 미침
  - 단점: 기술과 인간성 간의 갈등 간과, 인적 자원의 부족
  - 예시: 기업업무절차혁신 등을 통한 행정조직 내의 운영과정 및 일의 흐름을 개선, 행정전산망 등 장비, 수단의 개선, 행정과정의 새로운 분석기법의 적용

 * 인관관계론적 접근방법 - 조직발전(OD)
  - 개념
   = 인간행태의 변혁이 조직구조의 변화를 초래하고 새로운 관리기법을 적용 가능하게 한다는 생각에 근거를 둠
   = 행정인을 조직의 중요 의사결정에 참여시킴으로써 구성원들에게 조직목표를 더욱 절실히 이해하게 함
   = 여러가지 조직발전기법과 목표관리를 활용함으로써 구성원들의 자율적인 행태변화를 유도하고 그 결과를 바탕으로 행정계획을 도모함
   = 행정인의 가치관, 신념, 태도를 인위적으로 조작적 전략이나 실험실 훈련 등에 의하여 변혁시켜 행정체제 전체의 개혁을 도모하는 접근방법
  - 문제점
   = 인간의 행태 변화에는 오랜 시일 소요
   = 자유로운 의사소통과 토의가 받아들여지기 어려운 권위주의적 행정문화 속에서는 성공하지 어려움
   = 정부조직의 경우 현실적인 법적 제약이 존재
  - 예시: 집단토론, 감수성훈련

 * 종합적 접근방법
  - 개념: 구조적 접근방법, 관리-기술적 접근방법, 인간관계록 접근방법을 상호보완적으로 조화-융합하려는 접근방법
 
 * 행정개혁 저항요인
  - 기득권의 침해
  - 개혁내용의 불확실성
  - 피개혁자의 능력 부족
  - 관료제의 경직성과 보수적 경향
  - 참여 부족과 국민의 무관심
  - 비공식적 인간관계의 과소평가
  - 계선과 참모 간의 불화
  - 정치적, 사회적 요인
  - 타성으로 인한 저항
  - 고객집단의 저항

 * 저항의 극복방안
  - 강제적 전략
   = 상급자의 권한 행사
   = 물리적인 제재나 압력 사용
   = 의식적인 긴장 조성
  - 공리적, 기술적 전략
   = 개혁의 선택적 추진, 점진적 추진
   = 적절한 시기의 선택
   = 개혁안의 명확화
   = 개혁방법, 기술의 수정
   = 적절한 인사배치
   = 공공성의 강조
   = 손실의 방지, 최소화, 손실의 보상
  - 사회적, 규범적 전략
   = 참여의 확대
   = 커뮤니케이션의 촉진
   = 교육훈련의 실시
