03. 중세의 우리 역사
1> 중세의 정치
 * 후삼국의 통일 과정
  - 후백제 건국: 견훤이 무진주를 점령하고 완산주를 도읍으로 하여 건국(900)
  - 후고구려 건국: 양길의 수하인 궁예는 송악을 도읍으로 후고구려 건국(901)
   = 후에 도읍을 옮기고 국호와 연호를 여러 번 고침
   = 왕건의 활약으로 전라도 금성(나주)를 점령하여 후백제 견제
  - 고려의 건국: 송악의 호족 왕건은 궁예의 폭정에 반발하여 918년 궁예를 몰아내고 철원에서 고려 건국
   = 919년 철원에서 다시 송악으로 도읍을 옮겼다.
  - 후삼국의 통일
   = 926년 발해가 멸망하고 그 유민들을 받아들임
   = 930년 고창 전투, 934년 운주성 전투에서 후백제에 승리하고 주도권 장악
   = 935년 6월 권훤이 후백제에서 귀순해 옴.
   = 935년 11월 신라 경순왕이 고려에 항복함
   = 936년 일리천 전투, 황산벌 전투에서 후백제를 정벌하고 후삼국 통일
   
 * 태조 (1대, 918~943. 연호: 천수)
  - 민생 안정 정책: 지나친 세금 징수 금지, 흑창을 설치하여 빈민 구제
  - 호족 통제 정책
   = 사심관 제도: 경순왕이 항복하자 신라의 옛 수도인 경주의 사심으로 삼음. 다른 공신들도 그 지역 출신의 사심으로 삼음, 건국 초 개경에 거주하는 지배 계급을 매개로 간접적인 지방 통제 꾀함
   = 기인 제도: 일종의 인질 제도. 왕권과 호족 쌍방 간의 호혜적 바탕 위에서 운용. 훗날 왕권이 강화되자 인질로서의 통제는 점차 약화됨.
  - 호족 회유 정책
   = 호족 회유: 호족들의 대우를 두텁게 하고, 중폐비사의 저자세로 대함.
   = 방법: 결혼 정책 및 사성 정책(왕씨 성 하사)를 추진. 공신들에게 역분전 지급
  - 북진 정책
   = 고구려 계승 의식: 발해 유민을 수용하고 고구려 계승 의식 공고히 함
   = 서경 중시: 서경을 북진 정책의 전진 기지로 삼음
   = 반거란, 반여진 정책: 반거란 정책을 표방하면서 북방 개척. 태조 말에는 청천강부터 영흥만까지 영토 확대
  - 통치 규범 정리: [정계]와 [계백료서]를 저술하고 훈요 10조 남김
  
 * 광종 (4대, 949~975)
  - 개혁 정치를 동한 왕권 강화
  - 개혁 방향: 혜종, 정종 시기의 왕권 불안 해소하기 위한 왕권 강화 정책.
   = 후주의 상기를 영입하여 개혁 정책 방향 제시.
   = 공신 세력을 대표했던 대상 준홍과 자승 왕동을 역모죄로 숙청
  - 노비안검법 (광종 7년, 956)
   = 실시: 후삼국 시대의 혼란기에 불법적으로 노비가 된 사람들을 조사하여 해방
   = 목적: 호족들에게 경제, 군사적 기반 약화를 위한 정책
  - 과거제 실시 (광종 9년, 958)
   = 실시: 후주에서 귀화한 쌍기의 건의로 실시
   = 목적: 무훈공신의 세력 약화, 신진 인사를 기용하여 왕권 강화 목적
  - 백관의 공복 제정 (광종 11년, 960)
   = 관리들의 복색을 자색, 단색, 비색, 녹색의 4색 공복을 제정
  - 창제 건원: '광덕', '준풍' 등의 연호를 사용해서 자주성 표현
  - 주현공부법: 주, 현 단위로 조세와 공물, 부역 등을 부과
  - 송과의 통교
  - 제위보(빈민 구제 기금) 설치
 * 성종 (6대, 981~997)
  - 체제 정비와 귀족 사회 성립
  - 최승로의 시무 28조를 채택
   = 유교적 정치 이념 구현
  - 성종의 정책
   = 중앙 통치 기구의 정비: 2성 6부제를 기반으로 태봉과 신라의 제도 참작
   = 지방 제도의 정비: 12목을 설치해 지방관 파견. 향직 제도 개편
   = 유교 교육 진흥을 위한 국자감 설치
 * 최승로
  - 시무 28조
   = 국방 강화
   = 유교적 정치 이념 구현
   = 지방관을 파견하여 중앙 집권 강화
   = 중국 문물의 수용
   = 연등회와 팔관회 축소
 * 도병마사(도평의사사)
  - 성격: 국방 문제를 당담하는 임시 기구
  - 최고 정무 기구: 고려 후기에 도평의사사로 개편됨
   = 구성원이 확대되고 국정 전반에 걸친 중요 사항을 담당하는 최고 정무 기구로 발전
 * 대간
  - 어사대의 관원은 중서문하성의 낭사와 함께 대간으로 불림.
  - 왕의 잘못을 논하는 간쟁, 봉박, 서경권을 가짐
   = 간쟁: 왕의 잘못을 논하는 일
   = 봉박: 잘못된 왕명을 시행하지 않고 되돌려 보냄
   = 서경권: 모든 관리 임명 및 법령의 개폐, 국왕의 대관식 등에 동의
 * 음서
  - 5품 이상 고위 관직자의 자손 등은 추천을 통해 관직에 오를 수 있는 제도
  - 관료로서의 지위를 세습함
  - 귀족의 관직 독점 기반
 * 묘청(서경 천도운동)
  - 인종 때 일어난 일(1135)
   = 비슷한 것으로 실권자인 이자겸의 난
  - 묘청의 풍수지리설을 근거로 서경으로 천도할 것을 주장
   = 서경으로 도읍을 옮겨 금나라를 정벌하자고 주장
  - 개경파의 반대로 받아들여지지 않자 서경에서 난을 일으킴
   = 국호: 대위국, 연호: 천개, 군대: 천견충의군
  - 김부식 등에게 진압됨
 * 무신 정변
  - 문벌 귀족 사회의 모순
   = 의종 역시 무신을 차별함
  - 1170년(의종 24년) 정중부, 이의방 등이 정변을 일으켜 다수의 문신을 죽이고 의종을 폐함
   = 무신은 관직을 독점하고 사병을 늘려감
  - 무신들의 권력 쟁탈전
   = 정중부(중앙) -> 경대승(도방) -> 이의민(중방) -> 최충헌(교정도감) -> 최우(정방, 서방)
   = 최충헌 이후 4대 60여년 동안 최씨 무신정권 시기 수립
 * 최충헌(1196~1219)
  - 사회 개혁책 제시
   = 봉사 10조와 같은 사회 개혁책 제시
   = 농민 항생 적극적으로 진압
  - 권력 유지에 치중: 사회 개혁잭은 흐지브지되고, 최충헌은 오히려 많은 토지와 노비를 차지함.
  - 교정도감의 설치(희종 5년, 1209): 최고 집정부의 구실을 하는 교정도감 설치
  - 도방 설치: 사병 기관인 도방을 부활시켜 신변 보호.
   = 삼별초와 함께 최씨 정권을 유지하는 군사적 기반이 됨.
 * 최우(1219~1249)
  - 정방 설치: 자기 집에 정방을 설치하여 모든 인사권 장악
  - 서방 설치: 문학적인 소양과 함께 행정 실무 능력을 갖춘 문신들을 등용하여 고문 역할을 맡게 함
  - 삼별초 조직: 최씨의 사병 조직
 * 삼별초
  - 좌별초, 우별초, 신의별초(신의군)으로 조직.
  - 군과 경찰 등의 공적 임무를 띈 최씨의 사병 조직
  - 대몽 강화 시기 때 몽골군과 끝까지 항쟁함.
   = 강화도에서 진도로, 그 후에는 제주도로 이동하여 망명
   = 여몽 연합군이 삼별초를 진압한 후 제주도에 탐라총괄부 설치됨.
 * 대외 항쟁
  - (거란, 여진, 몽골, 홍건적, 왜구)
 * 거란의 침입과 격퇴
  - 제 1차 침입(성종 12년, 993)
   = 원인: 고려의 거란에 대한 강경책과 광종 이후 송과의 수교 및 정안국의 존재
   = 경과: 거란의 소손녕은 고려가 차지한 고구려의 옛 땅을 내놓을 것과 송과의 단교를 요구하며 80만 대군을 이끌고 침입. 청천강에서 거란의 침입을 저지하는 한편, 서희가 거란과 협상함
   = 결과: 거란으로부터 고구려의 후예를 인정받고 압록강 동쪽의 강동 6주를 확보. 송과 교류를 끊고 거란과 교류할 것을 약속
  - 제 2차 침입(현종 1년, 1010)
   = 원인: 거란군이 퇴각한 뒤 고려는 계속 송과 친선하며 거란의 요구를 거절
   = 경과: 강조의 정변을 계기로 강동 6주를 넘겨줄 것을 요구하며 40만 대군으로 다시 침입함. 개경이 함락되어 왕이 나주로 피난했으나 거란군의 배후에서 양규가 선전
   = 결과: 거란군은 퇴로가 차단될 것을 두려워하여 고려와 강화하고 물러남
  - 제 3차 침입 (현종 9년, 1018)
   = 원인: 현종의 입조를 재차 요구하고 강동 6주를 반환할 것을 요구했으나, 고려가 모두 거절
   = 경과: 거란의 소배압이 10만 대군으로 침입해 옴. 개경 부근까지 침입한 거란족은 귀주에서 강감찬이 지취하는 고려군에게 대파됨. (귀주 대첩, 1019)
   = 결과: 양국은 강화를 맺고 고려는 거란의 연호를 썼으며, 강동 6주의 요구는 철폐됨.
 * 여진 정벌과 9성 개척
  - 여진과의 1차 접촉에서 패배(1104)
  - 별무반의 조직: 숙종대 윤관의 건의로 신의군(기병), 신보군(보병), 항마군(승병)으로 편성
  - 윤관의 여진 정벌(예종 2년, 1107) -> 동북 9성 개척
  - 금의 건국(1115, 아골타): 거란(요)와 북송을 멸망시키고 고려에 군신 관계를 요규
   = 이자겸은 사대 관계를 체결함 (묘청의 서경 천도와 이어짐)
 * 몽골과의 정책과 강화
  - 고려와 몽고의 첫 접촉(강동의 역, 1218)
  - 몽고의 침입
   = 제 1차 침입(1231): 몽고 사신 저고여의 피살 -> 살리타의 침압 -> 박서의 항전 -> 개경이 포위되어 강화 -> 몽고는 다루가치 설치 후 철수 -> 최우는 강화도로 천도함
   = 제 2차 침입(1232): 김윤후는 처인성(용인)에서 살리타 사살. 초조대장경이 소실
   = 제 3차 침입(1238): 황룡사 9층탑의 소실
   = 충주성 전투(1253): 몽골의 5차 침입 때 김윤후의 활약
  - 원과의 강화
   = 원종 11년(1270) 개경으로 환도 -> 몽골과의 강화
 * 충선왕 (1298, 1308~1313)
  - 정방의 일시적 폐쇄
  - 개혁기구: 사림원
  - 소금의 전매제 실행
  - 상왕으루 물러난 이후 북경에 만권당(학술 연구 기관) 설치
 * 공민왕의 개혁 정책
  - 원,명 교체기 이용
  - 권문세족 집권기의 사회적 혼란
  - 개혁 추진 방향: 대외적으로 반원 자주 실현, 대내적으로 왕권 강화 정책
  - 반원 자주 정책
   = 기철 등 친원 세력 숙청
   = 정동행성 이문소 폐지
   = 인당으로 하여금 요동 지방 공략
   = 쌍성총관부를 무력으로 수복
   = 몽골풍 폐지
   = 2성 6부제 복구
  - 권문세족 압박 정책
   = 신진 사대부를 억제하던 정방을 폐지하고 문관의 인사권을 이부로 환원
   = 신돈을 정민변정도감(1366)의 판사로 삼아 토지 및 노비 제도 개척
   = 유학 교육 강화(성균관)
   = 과거제 정비
  - 신돈과 공민왕이 제거되면서 중단
 * 위화도 회군
  - 개혁 방향의 갈등: 최영이 이성계를 대표로 하는 사대부 세력의 뒷받침을 받아 이인임 일파 축출. 하지만 개혁 방향에 갈등
  - 온건 개혁파
   = 이색, 정몽주, 이승인, 길재
   = 고려 왕조 안에서 점진적 개혁
   = 성리학만 정학으로 인정
   = 군사력 미비로 혁명파 제거에 실패
   = 조선 사림파(16세기 집권)에 계승
  - 급진 개혁파(혁명파)
   = 이성계, 정도전, 조준, 윤소종 등
   = 고려 왕조를 부정하는 역성 혁명 추진
   = 성리학의 정학 인정, 주례 수용
   = 신흥 무인 세력, 농민, 군사들과 협력
   = 조선 훈구파(15세기 집권)
  - 위화도 회군: 최영이 요동을 정벌하기 위해 군사를 보냈으나, 이성계는 위화도에서 회군하여 최영을 제거하고 군사적 실권 장악

2> 중세의 경제
 * 전시과 제도
  - 운영 원칙과 특징
   = 관리의 직역에 대한 대가로 지급. 수조권 부여
   = 전지(농토)와 시지(땔감) 지급
   = 전국을 대상으로 함
  - 역분전(태조, 940): 논공행상적 성격
  - 시정 전시과(경종 원년, 976): 관품의 상하와 함께 인품을 반영함
  - 개정 전시과(목종 원년, 998): 
   = 기준에서 인품 배제
   = 관등을 기준으로 18품계 차등 지급
   = 한외과 규정이 마련됨
   = 전현직 관리 모두 지급
   = 산직보다 현직을 더 우대
   = 무관보다 문관 우대
  - 경정 전시과(문종 30, 1076): 전시과 제도의 완성
   = 무반에 대한 대우가 현저히 상승
   = 산직자를 배제하고 현직자에게만 지급
   = 한외과 폐지
 * 고려 시대 토지의 종류
  - 공음전
   = 5품 이상의 관리들에게 과전 외에 지급된 토지
   = 세습 가능, 문벌 귀족의 지위 유지 기반
  - 한인전
   = 6품 이하 하급 관리의 자제로 관직에 오르지 못한 자에게 지급
  - 구분전
   = 자손이 없는 하급 관료와 군인의 유가족에게 지급
   = 생활 대책의 마련
  - 외역전
   = 향리에게 직역의 대가로 주어진 토지
  - 군인전(중앙 군인), 내장전(왕실의 경비 충당), 공해전(중앙과 지방의 각 관아의 경비 충당), 사원전(사원의 경비 충당ㄴ)
 * [농상집요]
  - 충정왕 1년(1394) 이암이 원나라로부터 소개하고 보급.
 * [주전도감]과 화폐 발행
  - 숙종 때 의천의 건의로 설치.
  - 은병(활구), 해동통보, 삼한통보, 삼한중보 등의 동전 주조
 * 관영 상점
  - 개경, 서경, 동경 등의 대도시에서는 관청의 수공업장에서 생산하는 물품을 판매
  - 상행위 감독 기구로 경시서 설치
  - 지방 시장에서는 자체적으로 시장을 개설하고, 행상인들이 활동함
 * 벽란도와 무역 활동
  - 통일 신라 시기 사무역은 고려 시대 때 통제를 받고 공무역이 장려됨
  - 송나라, 요나라, 일본, 아라비아 등과 교역
  - 예성강 어귀의 벽란도는 대외 무역의 발전과 함께 국제 무영항으로 번성
  

3> 중세의 사회
 * 각종 기금인 보의 등장
  - 목적: 공적인 기금을 만들어 이자를 공적인 사업의 경비로 충당
  - 종류:
   = 학보(교육 기금)
   = 광학보(승려들의 면학을 위함. 946년 정종 때 시행)
   = 제위보(빈민 구제 기금)
   = 경보, 금정보, 팔관보
  - 결과와 한계: 오히려 이자 취급에만 급급하여 농민들의 생활에 피해를 끼침
 * 중류층
  - 지배층과 피지배층의 중간 계급
  - 좁은 의미로 지배 기구의 말단 행정직 의미
  - 각 지방의 호족 출신은 향리로 편제
 * 향리
  - 지방관이 파견되지 않은 속현이나 부곡의 지배층
  - 공물 수취 등 지방 행정의 실무 담당
 * 호족
  - 통일 신라 출신의 호족들은 지방관인 향리로 편제되어 감
  - 고려 초에는 성씨를 사성 받아 자신의 근거지를 본관으로 인정받기도 함.
  - 호족 출신 향리는지방의 실질적인 지배층으로
   = 통혼 관계나 과거 응시 자격에 의해서도 하위인 향리와 구분
 * 문벌 귀족
  - 성격: 대대로 고위 관직을 차지하고 문벌 귀족 형성
  - 거주: 대대로 개경에 거주, 죄를 지은 자는 귀향함
  - 관직을 바탕으로 하여 토지를 직접함
  - 유력한 귀족 가문과 중첩된 혼인 관계
   = 왕실과 외척이 되기도 하였으며, 왕실의 외척이 되는 것은 권력을 장악하는 수단이 되기도 함.
  - 지방 향리의 자제들이 과거에 급제에 관료가 되기도 하고, 귀족이 낙향해 향리로 전락하는 경우도 있음.
 * 권문세족
  - 무신정권 붕괴와 함께 등장한 귀족
  - 정계의 요직을 장악하고, 막대한 농장을 소유함
  - 음서의 혜택을 이용해 가문을 세습함
  - 유형
   = 이전부터 이어진 세력가문
   = 무신정권 때 등장한 가문
   = 원의 세력을 배경으로 성장한 가문
  - 대규모의 농장을 소유하고, 조세를 면제받아 국가 재정에 막대한 피해
 * 신진 사대부
  - 출신: 과거 시험에 합격한 후 관계에 진출한 향리 출신들
  - 등장 시기
   = 무신정권기부터 등장 시작
   = 최씨 정권 시절에는 무신들의 학문과 행정 능력의 보완 구실이었음
   = 무신정권 붕괴 후 더욱 활발하게 중앙 정계로 진출하여 고려 말에는 커다란 사회 세력 형성
  - 전시과의 붕괴로 과전을 받지 못하게 되자 권문세족을 비판
  - 성리학의 수용: 권문세족의 친원적, 친불교적 성향 만대
  - 개혁의 추구: 권문세족 시기 사회의 여러 모순을 비판하고 사회 개혁과 혁신 추구
 * 여성의 지위
  - 재산의 자녀 균분 상속, 남편이 죽으면 재산의 분배권은 아내가 행사
  - 연령순으로 호적 기재, 아들딸이 돌아가며 제사를 지냄
  - 사위가 처가의 호적에 입적하거나 처가살이 하는 경우도 많음
  - 사위와 외손자에게도 음서의 혜택
  - 여성의 재가가 자유로웠으며, 자식의 사회적 진출에도 차별받지 않음.
 * 향, 부곡, 소 등의 특수 지역
  - 향과 부곡민은 농업에 종사
  - 소에 거주하는 사람은 수공업이나 광업품의 생산을 담당함
  - 법적으론 양인이었으나, 일반 양민보다는 규제가 심함
   = 거주 이전의 자유가 없었으며, 양민보다 세금을 많이 냄.
  - 일반 군현민들이 반란을 일으킬 경우, 고을 전체가 강등되기도 함
 * 민의 사회적 지위
  - 소유하고 있는 민전에 농사를 짓거나, 황무지를 개간하여 토지를 소유할 수 있었다.
  - 토지가 없는 경우 국, 공유지 등의 각종 토지를 빌려 소작 가능. 일정한 수확을 주인에게 줌
  - 조세, 공납, 역의 의무가 부과되었지만, 출세에 법적 제한은 없었음.
 * 무신 정권 시대 하층민의 봉기
  - 왕조 질서를 부정하는 성격에서, 지방관의 탐학을 국가에게 호소하는 성격까지 다양함
  - 김보당의 난, 조위총의 난: 개경에서 시작했으며, 무신정권 반대 성격
  - 망이-망소이의 난: 1176년, 공주 명학소에서 충순현으로 승격됨
  - 김사미-효심의 난: 신라 부흥 운동 표방(1193)
  - 만적의 난: 노비들의 신분 해방적 성격(1198, 최충헌 집권 이후)
  - 최광수의 봉기: 서경에서 고구려 부흥 표방 (1217)
  - 이연년의 봉기: 담양에서 백제 부흥 표방 (1237)


4> 중세의 문화
 * 관학 진흥 정책
  - 국자감
   = 유학 교육 + 기술학 교육(율, 서, 산학)
   = 신분별 입학(유학부: 국자학-3품, 태학-5품, 사문학-7품 이하 관리의 자제)
   = 기술학부(율, 서, 산학)은 8품 이하 하급 관리들의 자제 또는 평민도 입학 가능
  - 숙종: 국자감에 서적포 설치
  - 예종: 7재의 전문 강좌 설치, 양현교 설치
  - 인종: 경사 6학 정비
  - 충렬왕: 섬학전 설치, 경사교수도감의 설치, 공자 사당인 문묘를 새로 건립
  - 공민왕: 성균관을 순수 유학 교육 기관으로 개편
 * 9재 학당과 사학 12도
 - 문종 때 해동성자라 불린 최충이 은퇴 후 문헌공도를 비롯한 9재 학당 건립
  = 사학에서 교육을 받은 학생들이 과거에서 좋은 성적을 받자 국자감의 관학 교육은 위축됨 
 
 * 고려의 불교
  - 왕실, 귀족 불교
  - 유, 불 융합
  - 현세 기복적 신앙
 * 의천과 천태종
  - 문종의 넷째 아들로, 송에서 화엄과 천태의 교리 터득
  - 천태종은 교종(화엄종)을 중심으로 선종을 통합한 종파
   = 원효의 화쟁 사상을 중시
  - 천태종의 통합 이론으로 교관겸수 강조
   = 교학과 선을 함께 수행하되, 교학의 수련을 중심으로 선을 포용할 것
  - 배우고 나서 행동으로 옮길 것을 강조
 * 보조 국사 지눌과 수선사 결사 운동
  - 무신정권기의 불교
  - 지눌은 명리에 집착하는 당시 불교계의 타락상을 비판함
  - 독경과 선 수행, 노동에 고루 힘쓰자는 개혁 운동인 수선사 결사 제창
  - 송광사가 중심. 개혁적인 승려들과 지방민들의 적극적인 호응
 * 조계종, 선-교 일치 사상의 완성
  - 조계쌍수: 선과 교학이 근본에 있어 둘이 아니라는 사상
  - 돈오점수: 내가 곧 부처라는 깨달음을 위한 노력, 꾸준한 수행으로 깨달음의 확인
  - 선종을 중심으로 교종을 포용하는 논리는 고려 불교가 지향하던 선교 일치 사상 완성
 * 혜심과 결사 운동의 발전
  - 유불 일치설 주장
  - 성리학을 수용할 수 있는 사상적 토대 마련
 * 요세
  - 강진 만덕사에서 천태종 계열 결사 운동인 백련결사 제창.
  - 행동을 참회하는 법화 신앙에 중점을 둠
  - 수선사와 양립하여 고려 후기 불교계 이끔

 * 고려의 역사서
  - 역사 편찬 방법
   = 기천체: 본기, 세가, 열전, 지, 표로 구성. (사마의 [사기]의 구성)
   = 편년체: 년, 월, 일별로 사실 서술.
   = 기사 본말체: 사건 중심으로 서술.
   = 강목체: 강과 목으로 나누어 서술.
  - [삼국사기] (1145)
   = 우리나라 현존 최고의 역사서이자 대표적인 정사
   = 인종 때의 문하시중 김부식이 편찬
   = 서경파가 제거되고 문벌 귀족이 지배하던 역사 의식 반영 (유교적 합리주의 사관)
   = 구성: 본기, 연표, 지, 열전으로 구성
   = 본기에는 삼국 왕실 역사가 균형있게 기록되어 있으나, 연표, 지, 열전은 신라사에 편향됨
   = 신화, 설화, 불교, 도교, 풍수지리 관련 사항 등을 배제한 객관적이고 합리적 입장에서 기록. 자주적 역사 의식
   = 한계: 그러나 고조선, 부여, 발해를 배제하고 삼국 시대(특히 신라)에 한정하여 기록함.
  - [삼국유사] (충렬왕 7~9년, 1281~1283년 저술)
   = 선종 승려 일연이 저술
   = 원 간섭 초기로 자주적 사관
   = 사찬 사서이며, 설화를 중심으로 편집
   = 기사 본말체적 구성
   = 문벌 귀족의 유교적 합리성에서 탈피해, 신화, 설화, 불교, 도교, 풍수지리 관련 사항을 원형대로 수료
   = 내용마다 전거 자료를 제시하여 인용인지 자신의 견해인지 명확히 밝힘
   = 서술 범위: 고조선을 국가 기원으로, 단군을 민족 시조로 상정
   = 한계: 중국 쪽 자료를 더 신뢰하여 기자, 위만, 마한, 진한 등은 중국 계열로 분류. 또한 발해를 우리 민족사로 인정하지 못함
  - [제왕운기] (1287년 저술)
   = 이승휴가 저술
   = 상권에는 중국 금나라까지의 역대 사적을 7언시로 서술,
   = 하권에는 고조선부터 충렬왕까지의 사적을 통사적 형태로 7언시 또는 5언시로 수록
   = 대조영의 발해를 우리 역사 내로 흡수하여 서술한 것이 역사적 의의
   = 기자 조선을 인정하여 사대적 역사 인식, 을지문덕과 강감찬의 기록이 삭제됨.
 
 * 대장경의 편찬
  - 경, 율, 론의 삼장으로 구성되어 불교 경전을 집대성한 것
  - 문화적 의의가 높은 유산
 * [초조대장경]
